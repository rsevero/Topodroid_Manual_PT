Código geomorfolófico

Esta caixa de diálogo permite definir um código geomorfológico a ser associado a uma foto ou base ou ainda alterá-lo.

A lista de código geomorfológicos é lida do arquivo de texto "geocodes" que deve ser colocado na pasta privada do aplicativo.

Cada linha do arquivo "geocodes" deve conter:

  • um número inteiro que identifica a classe da geomorfologia.
  • a sequência de caracteres do código (sem espaços).
  • uma descrição do código (que pode ter espaços).

As classes de geomorfologia são apresentadas em cores diferentes. Portanto é aconselhável listar juntas as geomorfologias da mesma classe.

O código não é mostrado na interface, mas é armazenado no banco de dados e no arquivo de croqui "tdr".
