CROQUI: MEDIDA

Essa caixa de diálogo pode ser acessada após selecionar uma medida (visada ou auxiliar) e clicar no botão Editar na tela Desenho no modo "editar".

Propriedades da medida:

  • Distância (D), Azimute (A) e Inclinação (I)
  • Bases e Tipo de medida: < D > (para frente), < B > (contra) e < M > (manual)
  • Comentário
  • Direção no perfil retificado: "esquerda" (-1), "vertical" (0), or "direita" (+1)
  • Projeção escalonada no pefil retificado: barra de rolagem para definir extensão da visada no perfil retificado (só disponível durante a visualização de perfil retificado e se a configuração Especiais >> Dados topo >> Proj vis escala estiver habilitada) [T]
  • Marcar como "duplicada"
  • Marcar como "de superfície"
  • Marcar como "comentada"
  • Ocultar a partir da base: checkboxes para indicar a partir de qual base da visada a linha de trena deve ser oculta (só disponível para visadas)
  • Cor: só disponível para medidas auxiliares e se a configuração Especiais >> Dados medida auxiliar >> Cores aux estiver habilitada

A visada é apresentada em laranga se a medida estiver inconsistente com as outras medidas da mesma visada e em vermelho se for detectado algum indício de anomalia magnética.

O botão Gravar confirma as alterações e fecha a caixa de diálogo.

Um alerta será apresentado se, após a edição da visada, ela diferir de outra visada entre as mesmas bases por mais de um quarto de seu comprimento ou 15°.

Tanto o botão Cancelar do TopoDroidX quanto o botão Voltar do Android fecham a caixa de diálogo sem gravar as alterações realizadas.

Ocultando partes da linha de trena

Quando se tratar de visadas, são apresentados checkboxes para indicar a partir de qual base da visada a linha de trena deve ser ocultada. As bases a partir das quais a linha de trena será ocultada serão apresentadas em vermelho ou violeta no croqui. O ocultamento de partes da linha de trena pode ser desfeito na caixa de diálogo da base limítrofe.

Projeção de visadas no perfil retificado [T]

No perfil retificado as visadas são, normalmente, desenhadas para a esquerda, direita ou na vertical (ou não desenhadas em absoluto). A escolha entre essas opções é feita através das opções "esquerda" (-1), "vertical" (0), "direita" (-1) ou nenhuma opção marcada.

Quando uma visada é selecionada no perfil retificado uma barra horizontal amarela é desenhada a partir de seu ponto mediano. Arrastando a partir do ponto mediano ao longo desta barra é possível alterar rapidamente a direção da visada no perfil retificado.

Projeção escalonada de visada no perfil retificado [T]

Também é possível definir uma projeção escalonada de visadas no perfil retificado: um valor fracional entre -1,5 e 1,5 no lugar das três opções inteiras -1 (esquerda), 0 (vertical) e 1 (direita). Por padrão o fator de projeção das visdas é um valor inteiro com a parte fracional zerada. Através da barra de rolagem abaixo dos checkboxes das opções de direção padrão é possível alterar tanto a parte inteira quanto a fracional do fator de projeção.

A parte fracional do fator de projeção é zerado quando:

  • a direção no perfil retificado é alterada através de uma ação de multisseleção (exceto pela ação "espelhar")
  • a direção no perfil retificado é alterada usando a linha amarela central na tela Desenho
  • a direção no perfil retificado é alterada na caixa de diálogo da tela Dados topo
  • a direção no perfil retificado é definida pelo TopoDroidX

Para poder definir um fator de projeção fracional a configuração Especiais >> Dados topo >> Proj vis escala precisa estar habilitada. Como a maioria dos programas de desenho de mapas não tem suporte para uma projeção escalonada, essa opção é desabilitada por padrão.

Cor medidas auxiliares [T]

É possível atribuir uma cor a cada medida auxiliar. A cor é armazenada no banco de dados sendo, dessa maneira, compartilhada entre os vários croquis.

A funcionalidade de coloração de medidas auxiliares é controlada pela configuração Especiais >> Dados medida auxiliar >> Cores aux [T]: nenhuma, livre ou paleta.

A caixa de diálogo de cor tem um botão através do qual é possível remover a cor específica da medida auxiliar.
