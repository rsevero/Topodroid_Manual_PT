Entrada manual de dados

Se você não tiver um dispositivo Bluetooth é possível entrar dados manualmente.

O botão Adicionar da tela Dados topo abre a caixa de diálogo para a entrada de dados de uma medida.

Você deve cadastrar os nomes de bases e os dados (distância, azimute e inclinação).

É possível preencher somente os dados da contra visada deixando os campos das informações da visada em branco quando a opção de configuração Dados topo >> Dados medidas >> Contra visada [A] estiver habilitada.

Para cadastrar uma medida auxiliar use "." (ponto) ou "-" (hífen) como nome da base PARA.

Medidas EDTC podem ser incluídas em uma visada criando automaticamente medidas auxiliares correspondentes. Neste caso, as medidas EDTC se referem à base DE.

Um alerta é apresentado se, ao editar uma visada, esta diferir de outra visada já existente entre as mesmas bases em mais de um quarto da distância ou 15°.

Botões

  • Gravar: cria a medida e atualiza os nomes de bases incrementando o próximo nome de base a ser criado de forma que a caixa já esteja pronta para cadastrar a próxima visada.
  • OK: cria a medida na topo e fecha a caixa de diálogo.

Usando o Android para definir direção

O azimute e a inclinação podem ser definidos usando-se os sensores do dispositivo Android: [T]

  • : o eixo de referência é a direção do lado mais comprido do dispositivo. Um temporizador (de 10s por padrão) permite que você ajuste a posição do disposito antes da medição ser realizada.
  • : o eixo de referência é a normal da tela do dispositivo: olhe para a base PARA na tela e inicie o temporizador quando a base estiver no centro do cursor. Um alerta é apresentado se a sensibilidade do sensor estiver baixa ou houver uma falha no sensor.

EDTC

Se as medidas EDTC tiverem sido definidas, as respectivas medidas auxiliares são também inseridas. É possível também definir se as medidas EDTC definidas são relativas à base PARA ou à base DE.

Se a inclinação da visada for menor do que o valor definido na configuração Dados topo >> Dados medidas >> Lim inclin OLNS as medidas EDTC serão posicionadas no plano vertical: E na horizontal à esquerda da visada (ou seja, com o azimute igual ao da visada menos 90°), D na horizontal à direita da visada, T para cima e C para baixo.

Se a inclinação da visada for maior do que o valor definido na configuração Dados topo >> Dados medidas >> Lim inclin OLNS as medidas EDTC serão posicionadas no plano horizontal: E à oeste, D à leste, T ao norte e C ao sul.

Unidades

Os dados devem ser registrados na unidade de medidas definida na configuração Dados topo >> Unidades >> Unidade distância: metros ou pés.

Modo mergulho

No modo mergulho os dados que definem as visadas são profundidade, azimute e distância. Trata-se do azimute fornecido como se obtido através de uma bússola comum com a graduação crescendo em sentido horário. Os dados de azimute são convertidos como se tivessem sido obtidos com uma bússola espeleológica (bússola de geólogo). Por exemplo, um valor de 340° é convertido para 20°.

O modo "mergulho" é controlado pela configuração Especiais > Dados topo > Modo mergulho.
