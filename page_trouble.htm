Resolução de problemas com dispositivos de medidas

Pareamento de dispositivo

Antes que o TopoDroidX possa se conectar a um dispositivo, este deve ser pareado com o Android. Você pode parear o Android com o DistoX ativo através do menu parear ou pode usar o aplicativo Configurações para parear a DistoX.

Se o Android pedir o PIN do DistoX, digite 0000 (quatro zeros). Para o PIN do SAP5, digite 000000 (seis zeros). O Android não deve pedir o PIN para o BRIC4, BRIC5 ou DistoXBLE.

Uma vez pareado com o DistoX, o Android deve manter o pareamento e não pedir o PIN novamente. Se ele continuar pedindo o PIN toda vez que você baixar dados do DistoX, tente alterar a configuração Dispositivos > Soquete DistoX para a opção inseguro.

Se você tiver problemas para conectar e baixar os dados com um dispositivo DistoX, tente habilitar a configuração Dispositivos > Gambiarra Z6 DistoX.

DistoXBLE conectando mas não baixando dados

Às vezes, o DistoXBLE pode conectar ao Android mas não baixar os dados. Isso pode acontecer imediatamente após ter habilitado o Bluetooth do Android. A solução é desligar e religar a DistoXBLE. O TopoDroidX deve reconectar automaticamente e começar a baixar os dados.
