Manual do Usuário do TopoDroid

marco.corvi@gmail.com


Topodroid é um app Android para mapeamento de cavernas. O Topodroid recebe dados brutos, ajuda a organizá-los e exibe a linha de trena imediatamente. Você pode desenhar o croqui com base na linha de trena durante a exploração reduzindo, dessa maneira, a probabilidade de erros no croqui, ajudando a detectar erros nos dados ainda dentro da cavidade. O Topodroid tem diversas funcionalidades que lhe auxiliam a ser mais produtivo dentro da caverna. Dessa maneira você tem um mapa praticamente pronto ao final da saída de exploração.

O Topodroid torna o tempo dedicado ao trabalho dentro da caverna mais eficiente. Sua função é ajudar o explorador a coletar dados com mais acurácia e tornar a exploração mais agradável, não se propõe a produzir um mapa finalizado da caverna. Para isso há bons programas para PC voltados para o trabalho no escritório.

O Topodroid tem cinco diferentes níveis de funcionalidades. Os níveis mais baixos já contam com recursos para fazer download de dados, administrar nomes de bases e desenhar croquis simples. Os níveis mais avançados podem ser usados para ajustar dados e croquis para simplificar o trabalho no escritório.

O Topodroid foi pensado para trabalhar com uma topo por vez, não para administrar projetos complexos de mapeamento que incluam diversas topos diferentes. Ainda assim conta com o Gerenciador de Projetos Therion que ajuda a preparar projetos mais complexos para serem posteriormente processados no Therion compondo diversas topos em um projeto e um Visualizador 3D com o qual você pode ver uma representação 3D de uma topo ou projeto.

O Topodroid funciona com a DistoX (versões 1, 2 e BLE), SAP5 e BRIC4 além de permitir o cadastro manual de dados. Observe que os aparelhos Android (celulares, tablets, etc.), assim como canetas ativas podem interferir nas medidas produzidas por esses dispositivos eletrônicos se estiverem suficientemente próximos.

Diretório privado do TopoDroid

O diretório privado do Topodroid (Android/data/com.topodroid.TDX/files) contém os diretórios com os símbolos de desenho (pontos, linhas e áreas) assim como o banco de dados device10.sqlite com os dados de calibrações, o diretório dos arquivos de calibração (diretório ccsv), os arquivos das firmwares DistoX2 e DistoXBLE (diretório bin) e os arquivos de cópia das memórias da DistoX (diretório dump).

Esse diretório e seus conteúdos são deletados quando o app é desinstalado a não ser que você marque a opção para manter os dados do app.

Diretórios de topos TopoDroid

O diretório base do Topodroid é Documents/TDX criado no armazenamento interno do aparelho Android. Contém o subdiretório central Topodroid, onde está o banco de dados distox14.sqlite, o diretório zip, o diretório tmp para os arquivos temporários assim como um diretório diferente para cada topo criada.

Esse diretório e seus subdiretórios não são apagados quando o app é desinstalado.

Dados da topo

Os dados da topo assim como os croquis são guardados no subdiretório Topodroid dentro de TDX. Cada topo tem um subdiretório dentro de Topodroid. A pasta Topodroid contém o banco de dados distox14.sqlite, os subdiretórios das topos e o diretório dos projetos Therion thconfig.

Concedendo permissões de acesso

A partir do Android 11, o Topodroid pode não ser bem sucedido em criar o diretório base TDX ou algum dos subdiretórios abaixo dele por não ter permissões de acesso suficientes ao dispositivo externo principal de armazenamento do aparelho. Nesse caso, você pode conceder permissões usando o adb (Android Debug Bridge):

  1. faça o download do adb em seu PC
  2. dê um toque em Configurações > Sobre o telefone, dê um toque em Informações do Software e tecle sete vezes em Número da compilação
  3. retorne a tela anterior, dê um toque em Opções do desenvolvedor e selecione Depuração de USB
  4. no PC abra o servidor adb. Entre o comando adb start-server
  5. conecte o Android ao PC com um cabo USB. Pode ser necessário permitir adb-connection do seu PC para o seu Android
  6. no PC entre esses três comandos:
    • adb shell appops set com.topodroid.TDX READ_EXTERNAL_STORAGE allow
    • adb shell appops set com.topodroid.TDX WRITE_EXTERNAL_STORAGE allow
    • adb shell appops set com.topodroid.TDX MANAGE_EXTERNAL_STORAGE allow

Importando topos antigas

O TopodroidX (versão 6) pode ser instalado junto com a antiga versão 5.1.40 pois usam diretórios base diferentes.

Antigas topos criadas na versão 5.1.40 do Topodroid podem ser importadas na versão 6 através de arquivos zip criados no Topodroid antigo.

Estes arquivos zip podem também ser criados com os scripts disponíveis em http://marcocorvi.altervista.org/caving/speleoapps/speleoapks/TopoDroidZipScript.html

Obs: o Topodroid versão 5.1.40 não é capaz de importar arquivos zip criados pela versão 6 devido a mudanças no banco de dados.

Recursos de hardware

O Topodroid é capaz de utilizar os seguintes recursos de hardware: bluetooth, câmera, microfone, GNSS e multitoque. As funcionalidades do Topodroid que dependam de um recurso não disponível em seu aparelho Android são automaticamente desabilitadas.

A conexão de internet só é utilizada para instalar traduções deste manual e para verificar se há novas versões disponíveis do app.

Bibliotecas nativas

O Topodroid contém algumas bibliotecas compiladas nativamente para algumas poucas arquiteturas: arm64_v8a, armeabi-v7a e x86_64. São as seguintes bibliotecas: powercrust e o envólucro tiff da libtiff. Esta última contém a libjpeg e a libpng lincadas estaticamente.

Erros e panes

Dados os poucos recursos disponíveis para o seu desenvolvimento, o app não pode ser testado em sua integralidade a cada nova versão. Guarde um backup do arquivo apk de instalação e do banco de dados das topos antes de atualizar o app.

  • Panes após selecionar uma opção pode ser relacionada a um problema na tradução. Repita a ação usando a versão em inglês do app. Se a pane desaparecer, entre em contato com o responsável pela tradução..
  • Se encontrar um erro ou o app não se comportar conforme esperado, envie um email para marco.corvi@gmail.com Inclua em sua mensagem as versões do app e do Android.
  • Para outras questões e sugestões de novas funcionalidades, envie um email para a lista de email do Topodroid: <topodroid@googlegroups.com>.
