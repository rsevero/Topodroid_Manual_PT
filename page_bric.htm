GUIA DE REFERÊNCIA BRIC4

Este é um resumo das funcionalidades do BRIC4.

O Manual do Usuário oficial pode ser encontrado em http://www.caveexploration.org/gear/bric4

Para ligar o BRIC4 aperte o botão externo três vezes em sequência. Para desligar, pressione o mesmo botão por 3 segundos. O BRIC4 se desliga automaticamente após decorrido o tempo definido na configuração Idle off [padrão 90s].

BOTÕES INTERNOS
  • Menu: abre o menu de opções
  • Para cima: move para a próxima opção a cima
  • Para baixo: move para a próxima opção a baixo
  • Back: retorna para a tela anterior
MENU

O menu principal é acessado através do botão interno Menu. Um clique no botão externo encerra a apresentação do menu e retorna o BRIC4 para o modo Medida. Você também pode retornar ao modo Medida apertando repetidamente o botão interno Back.

No modo Menu os botões internos tem as seguintes funções:

  • Enter: confirma a escolha realizada
  • Para cima: move para a próxima opção a cima
  • Para baixo: move para a próxima opção a baixo
  • Back: retorna para a tela anterior
Options
  • Dist: "Meters" (metros) ou "Feet" (pés)
  • Temp: "Celsius" ou "Fahrenheit"
  • Shot delay: atraso entre o clique no botão externo e a realização da medida. Entre 0 e 5 segundos. [padrão 0s]
  • Charge current: 100 mA ou 500 mA. [padrão 500 mA]
  • Err Sens: variação aceita para detecção de erros. De 0.2° até 2.8° em passos de 0.2°. [padrão 1.0°]
  • Idle off: limite de tempo de inatividade do BRIC4 após o qual se desliga sozinho. Em passos de 30s até 1800s. [padrão 90s]
Error Info

Apresenta os erros detectados nas últimas 10 medidas.

Calibration
  • Display report: apresenta relatório da última calibração realizada
  • Loop test: ferramenta para realizar teste de fechamento de poligonal
  • CAL Quick AZM: calibração rápida de azimute
  • CAL Full INC&AZM: calibração completa similar à da DistoX
  • CAL Rangefinder: calibração da medida de distância
Set Clock

Mova para baixo de Second para gravar.

  • Year: ano
  • Month: mês, de 01 a 12
  • Date: dia do mês, de 01 a 31
  • Hour: hora, de 00 a 23
  • Minute: minuto, de 00 a 59
  • Second: segundo, de 00 a 59
Bluetooth
  • Name: nome, por exemplo "Bric4_0039"
  • MAC address: endereço MAC, por exemplo "D52FDE2E98DE"
  • Connected to: conectado à, endereço MAC do dispositivo ao qual este BRIC4 está conectado
  • Reset BLE: reiniciar conexão Bluetooth BLE
  • Advanced Menu: abre o menu BLE Advanced
BLE Advanced

O menu BLE Advanced apresenta três opções:

  • RST to AT mode
  • RST to Run Mode
  • Curr Comm: CPU

O menu BLE Advanced apresenta as seguintes informações:

  • CPU Tx: a contagem de bytes transmitidos via BLE
  • CPU Rx: a contagem de bytes recebidos via BLE
  • BLE Sync: a situação atual dos dados: quantidade total de medidas na memória e quantidade de medidas enviadas
  • BLE Cmd: o último comando remoto BLE
Display
  • BL light: nível da iluminação de fundo, de 0 a 5
  • Color: "White" (branco), "Red" (vermelho), "Blue" (azul), "Green" (verde), "Purple" (roxo) e "Cyan" (ciano)
  • Contrast: nível de contraste, de 0 (invisível) a 30 (completamente escuro) [16]
Advanced Menu
  • Memory Clear: abre a tela Memory Clear
  • Sensor Raw Data: apresenta os valores brutos dos sensores
  • Firmware CPU: versões da firmware principal e do hardware além de opção para entrar no modo USB Bootloader
  • Firmware BLE: versão da firmware BLE além da opção Update para atualizar essa firmware
  • Cal Settings:
  • Redo Full Cal: recalcular os valores relativos a calibração completa
  • Redo AZM Cal: recalcular os valores relativos a calibração rápida (somente azimute)
Memory Clear

Apresenta a quantidade de medidas em memória e a quantidade de medidas enviadas. Apresenta opções para retornar Cancel e para limpar a memória Clear Memory, reiniciando o contador de medidas para 0.

Códigos de erro

O BRIC4 informa até dois erros para cada medida realizada. O código de erro é registrado junto com o valor do erro no comentário da medida. Nada é registrado no comentário se não for detectado um erro (código 0).

  1. nenhum erro
  2. valor do primeiro acelerômetro alto value [normal 1]
  3. valor do segundo acelerômetro alto value [normal 1]
  4. valor do primeiro magnetômetro alto value [normal 1]
  5. valor do segundo magnetômetro alto value [normal 1]
  6. disparidade entre os acelerômetros no eixo indicado disparity (1:X, 2:Y, 3:Z)
  7. disparidade entre os magnetômetros no eixo indicado disparity (1:X, 2:Y, 3:Z)
  8. erro de cálculo na trena (distância)
  9. sinal fraco na trena (distância)
  10. sinal excessivamente intenso na trena (distância)
  11. erro na identificação de padrão na trena (distância)
  12. limite de tempo esgotado na trena (distância) timeout
  13. erro na trena (distância) error
  14. identificador de mensagem errado na trena (distância) message id
  15. disparidade nas medidas de inclinação disparity
  16. disparidade nas medidas de azimute disparity
