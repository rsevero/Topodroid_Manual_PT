CORES TopoDroidX

Tela DADOS TOPO

Nome da topo na barra superior

Política de nomeamento automático de bases:

  • Visada [padrão]: azul
  • Contra visada: azul escuro
  • Contra visada + visada: violeta
  • Tripé: rosa
  • Anomalia magnética: vermelho
  • TopoRobot: laranja

Dados topo

Cor do texto:

  • branco: visada
  • cinza: medida repetida de visada
  • azul: medida auxiliar
  • verde: medida auxiliar de corte
  • azul escuro: medida auxiliar horizontal
  • azul profundo: medida auxiliar vertical
  • laranja: medida sem base
  • violeta:
  • outras: cores definidas pelo usuário

Cor de fundo:

  • preto: normal
  • vermelho: medida pouco confiável
  • cinza: medida com comentário
  • laranja: medida curta

Bases

Cor do texto:

  • verde: base selecionada
  • mesma cor do texto dos dados da topo: base não selecionada

Cor de fundo:

  • preto: normal
  • verde escuro: dados mais recentes
  • cinza: multisseleção
  • amarelo escuro: contra visada DistoX

Visadas

  • branco: normal
  • vermelho: não confiável
  • laranja: inconsistência com medidas irmãs

Medidas auxiliares

  • azul: normal
  • verde escuro: comentada
  • verde, azul escuro e azul profundo: classes de medidas auxiliares
  • outras: cores definidas pelo usuário

Tela DESENHO

Bases:

  • rosa: normal
  • verde: base selecionada
  • vermelha: linha de trena oculta após esta base
  • violeta: linha de trena oculta antes desta base
  • amarelo: base gravada

Tela DADOS CALIBRAÇÃO

dados de calibração

Cor do texto:

  • azul e laranja: grupos de dados
  • cinza: dados não usados

Cor do fundo:

  • violeta: dados parcialmente baixados (não usados)
  • vermelho: erro acima de 1 grau
  • verde: grande diferença da média do grupo
  • marrom: dados de sensores saturados
  • cinza: dados apagados (grupo -1)
