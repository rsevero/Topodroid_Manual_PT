BASE SELECIONADA

A base selecionada é a base a partir da qual a topo vai continuar. Se você selecionar uma base, o TopoDroidX vai continuar a criar bases a partir dela. Se não houver uma base selecionada, a topo continuará da última base criada.

Dessa maneira você pode começar um novo braço a partir do meio da linha de trena selecionando a base a qual esse novo braço deve se conectar. Como uma alternativa você pode também editar a base DE da nova visada após esta ter sido criada.

A base selecionada é destacada em verde tanto na tela Dados topo quanto na tela Desenho. Um clique longo em outra base faz com que esta outra base seja selecionada. Um clique longo na base selecionada desseleciona a base.

Uma base pode ser selecionada de três maneiras:

  • com um clique longo no nome da base na tela Dados topo;
  • na caixa de diálogo Bases gravadas clicando no botão Selecionar;
  • na caixa de diálogo Editar base na tela Desenho.
