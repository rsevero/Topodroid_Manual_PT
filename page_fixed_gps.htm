Coordenadas GPS

Essa caixa de diálogo é apresentada ao se clicar o botão GPS na tela Geopontos.

Se a opção de configuração Dados topo > Localização > App GNSS estiver em alguma opção diferente de nenhum o título da página será alterado para Geopontos [NOME DO APP GNSS]. Veja mais informações sobre os Apps GNSS.

Para usar o GPS do seu dispositivo o serviço de "Localização" do Android deverá estar habilitado. Por razões de segurança um app genérico como o TopoDroidX não pode habilitar o serviço de localização do Android, você precisa fazer isso manualmente. Se não estiver habilitado, o TopoDroidX apresenta uma caixa de diálogo pedindo para que o serviço seja habilitado (através do app "Configurações" do Android). Lembre-se de desabilitar esse serviço após o uso para poupar bateria.

Campos:

  • Base: nome da base a ser georreferenciada
  • Comentário: opcional.

Botões:

  • GPS: inicia/para a obtenção de coordenadas através do GPS.
  • Mapa: apresenta o ponto obtido em um app de mapeamento disponível no dispositivo Android.
  • Gravar: grava o ponto obtido.

Enquanto estiver buscando por satélites, o botão "GPS" permanecerá vermelho (enquanto não tiver conseguido estabelecer a posição atual) ou verde (quando tiver estabelecido a posição atual) e indicará a quantidade de satélites usados no momento. Quando uma quantidade suficiente de satélites tiver sido localizada o botão fica verde e as coordenadas obtidas são apresentadas abaixo. Você deve esperar até que os valores das coordenadas estejam estáveis e a acurácia da localização seja apresentada. Nesse momento deve parar a obtenção de localização GPS clicando novamente no botão GPS.

A acurácia informada é definida pelo hardware do dispositivo Android e das condições ambientais no momento da obtenção da localização.

No Android com API anterior à versão 26 somente a acurácia horizontal é informada. No Android com API igual ou superior à versão 26 tanto a acurácia horizontal como a vertical são apresentadas.

Configurações disponível em Dados topos >> Localização:

  • Localização precisa: tempo, em segundos, de espera para obtenção de localização mais precisa.
