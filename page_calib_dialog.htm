CALIBRAÇÕES DistoX

A caixa de diálog é acessada através do botão Calibração na tela Dispositivos.

BOTOẼS
  • Nova: abre a tela Info calibração onde você pode carregar dados para uma nova calibração e salvá-la
  • importar: importa uma calibração de um arquivo "csv"

Abaixo dos botões é apresentada uma lista das calibrações do dispositivo ativo.

Abrir uma calibração

Clique em uma calibração da lista para abrí-la na tela Info calibração.

Uma calibração já salva pode ser aberta na tela Dados calibração onde é possível baixar dados de calibração, calcular novos coeficientes de calibração e enviá-los para a DistoX.
