Principais

  • Diretório trabalho padrão indefinido (ou seja "TopoDroid").
  • Tam texto: tamanho do texto nos itens das listas. >= 1. [14]
  • Tam botões: {pequeno, normal, médio, grande, enorme}. [normal]
  • Nível recursos: {básico, normal, avançado, especialista, teste}. [normal]
  • Teclado. habilitar teclados especiais no lugar do teclado padrão do Android.
  • Sem cursor: desabilitar o curso do teclado especial do TopoDroidX
  • Idioma manual: traduções disponíveis: {inglês, espanhol, italiano, russo, húngaro, francês, português, chinês}. Para alterar essa configuração para alguma opção diferente de inglês o TopoDroidX depende de uma conexão de internet.
  • Idioma app {padrão, inglês, espanhol, italiano, russo, húngaro, francês, português, chinês}. [padrão] (ou seja, idioma do Android)
  • Orient tela: (indefinida, retrato, paisagem) [indefinida]

Importar/Exportar

  • Form export dados: formato padrão de exportação de dados. {nenhum,cave,cav,csv,csx,dat,dxf,grt,gtx,srv,sur,svg,svx,th,top,tro} [nenhum]
  • Form export croqui: formato padrão de exportação de croqui. {nenhum,th,csx,dxf,svg,xvi} [nenhum]
  • Form export aut croqui: formato de exportação automática de croqui. {nenhum,th,csx,dxf,svg,xvi} [nenhum]
  • Âng aux EDTC: ângulo mínimo entre uma medida auxiliar e uma visada para que uma medida auxiliar seja considerada EDTC (Esquerda/Direita/Teto/Chão) na exportação. [0]
  • Incl mín TC: inclinação mínima nas medidas TC (Teto/Chão). [0°]
  • Incl máx ED: inclinação máxima das medidas ED (Esquerda/Direita). [90°]
  • Espaçamento ponto: distância máxima entre pontos em uma linha Bezier exportada.

Custom import/export

  • Zipped symbols files: store/load symbol files to/from zip archives
  • Import datamode
  • Auto X-sections: insert X-Sections in sketch exports
  • Stations: include stations in sketch exports
  • LRUD splay count: number of splays used for LRUD [default false]
  • Zip-share category: categorized zip-share [default false]
If LRUD count is false all splays (within tolerances) are used to compute LRUD. The transverse dimension is set to the maximum value of the splay projections on the horizontal or vertical axes. An in-tolerance splay can be taken into account for both horizontal and vertical dimensions.
If LRUD count is true only the first four splays that satisfy the tolerances are used, and an in-tolerance splay is used only for the vertical or the horizontal dimensions. Each dimension is set to the value of the first acceptable splay. A dimension can be left with value zero by shooting a splay in a direction already set.
By default the tolerances are very permissive: 0° for vertical directions and 90° for horizontal.

Zip sharing can be categorized to restrict its handling to specific apps. If there is no app that can handle sharing, the action fails.

Settings import/export

Settings can be exported to the file "settings.txt" (in TopoDroid folder). This is a text file listing the values of all the settings.
Settings can be imported from the same file. It is possible to choose whether to import all the settings or only the functional ones.
If there is a problem parsing the file, the settings are not changed.
Some interface settings (like the buttons size) need to restart the app to become effective.
