Principais

  • Tam texto: tamanho do texto dos itens das listas. >= 1. [14]
  • Tam botões: {pequeno, normal, médio, grande, enorme}. [normal]
  • Nível recursos: {básico, normal, avançado, especialista, teste}. [normal]
  • Idioma manual: traduções disponíveis: {inglês, espanhol, italiano, russo, húngaro, francês, português, chinês}. Para alterar essa configuração para alguma opção diferente de inglês o TopoDroidX depende de uma conexão de internet.
  • Idioma app {padrão, inglês, espanhol, italiano, russo, alemão, português, chinês, húngaro}. [padrão] (ou seja, idioma do Android)
  • Orient tela: (indefinida, retrato, paisagem) [indefinida]

Importar/Exportar

  • Form export dados: formato padrão de exportação de dados. {nenhum,cave,cav,csv,csx,dat,dxf,grt,gtx,srv,sur,svg,svx,th,top,tro} [nenhum]
  • Form export croqui: formato padrão de exportação de croqui. {nenhum,th,csx,dxf,svg,xvi} [nenhum]
  • Form export aut croqui: formato de exportação automática de croqui. {nenhum,th,csx,dxf,svg,xvi} [nenhum]
  • Âng aux EDTC: ângulo mínimo entre uma medida auxiliar e uma visada para que uma medida auxiliar seja considerada EDTC (Esquerda/Direita/Teto/Chão) na exportação (arquivos Compass/TopoRobot/Winkarst/Polygon/VisualTopo). [0]
  • Incl mín TC: inclinação mínima nas medidas auxiliares TC (Teto/Chão). [0°]
  • Incl máx ED: inclinação máxima das medidas auxiliares ED (Esquerda/Direita). [90°]
  • Espaçamento ponto: distância máxima entre pontos em uma linha suave (Bézier) exportada.

Habilitar

Define quais opções de formato de exportação estão disponíveis na caixa de diálogo de exportação. Formatos suportados:

  • Compass (DAT)
  • cSurvey (CSX)
  • GHTopo (GTX)
  • Polygon (CAVE)
  • Survex (SVX)
  • Therion (TH)
  • Topo (CAV)
  • TopoRobot (TAB)
  • VisualTopo (TRO, TROX)
  • Walls (SRV)
  • Winkarst (SUR)
  • CSV
  • DXF
  • KML
  • GPX
  • Shapefile

Importar

  • Mapa cor PocketTopo: definição do mapeamento entre cores das linhas PocketTopo e tipos de linhas Therion.
  • Estimar ref retif perfil: estimar referência para retificação de perfil na importação de arquivos Compass e VisualTopo. [sim]

Survex

  • Term linha: tipo de terminação de linha a ser usado nos arquivos exportados {Linux,Windows}. [Linux]
  • Base aux: incluir às medidas auxiliares exportadas em formato Survex um nome de base PARA cujo nome é composto da base DE e uma letra adicionada. [não]
  • EDTC: incluir medidas EDTC (Esquerda/Direita/Teto/Chão) nos arquivos Survex e Therion exportados. [não]

Therion

  • Criar arq config Therion: criar arquivo thconfig na exportação Therion. [não]
  • Mapa Therion início: incluir as menções aos mapas Therion no início dos dados da topo. [não]
  • Med aux therion: adicionar linhas u:splay aos arquivos th2. [não]
  • EDTC: incluir medidas EDTC (Esquerda/Direita/Teto/Chão) nos arquivos Survex e Therion exportados. [não]
  • Escala Therion: escala de exportação de arquivos th2/xvi. [1:100]
  • Imagem XVI: incluir chamada no arquivo th2 para imagem XVI. [não]

cSurvey

  • Pref base: incluir prefixo nos nomes das bases na exportação de arquivos Compass, cSurvey e WinKarst. [vazio]

Compass

  • Pref base: incluir prefixo nos nomes das bases na exportação de arquivos Compass, cSurvey e WinKarst. [vazio]
  • Aux Compass/Walls: exportar medidas auxiliares em arquivos Compass/Walls ou como 12 12-1 ou como 12 -.
  • Inverter Esq/Dir: inverter as medidas auxiliares esquerda e direita nos arquivos Compass exportados. [não]

VisualTopo

  • Aux VisualTopo: incluir medidas auxiliares em arquivos VisualTopo. [não]
  • EDTC em DE: medidas auxiliares ETDC na base DE em arquivos VisualTopo. [não]
  • Form Trox: exportação VisualTopo cria arquivos trox no lugar de arquivos tro. [não]

Walls

  • Med aux Walls: incluir medidas auxiliares nos arquivos Walls exportados. [não]

SVG

  • Circuito SVG: exportar arquivos SVG preparados para circuito. [não]
  • Grade: incluir grade no arquivo SVG/XVI exportado. [não]
  • Orient linha: incluir indicador de orientação de linha nos arquivos SVG exportados. [não]
  • Med aux SVG: incluir medidas auxiliares nos arquivos SVG exportados. [não]
  • Espess ponto: espessura da linha usada para desenhar os símbolos de pontos nos arquivos SVG exportados. [0.5]
  • Espess rótulo: espessura da linha usada para os textos dos rótulos nos arquivos SVG exportados. [1.5]
  • Espess linha: espessura da linha nos arquivos SVG exportados. [2.5]
  • Espess grade: espessura da linha usada para desenhar a grade nos arquivos SVG exportados. [2.5]
  • Espess medida: espessura da linha usada para desenhar as visadas nos arquivos SVG exportados. [2.5]
  • Tam orient linha: tamanho da indicação de orientação de linha usada nos arquivos SVG exportados. [12]
  • Tam base: tamanho do texto usado para nomear bases nos arquivos SVG exportados. [32]
  • Tam rótulo: tamanho do texto usado para rótulos nos arquivos SVG exportados. [48]
  • Programa alvo: qual programa usado como modelo na exportação {Inkscape, Illustrator} [Illustrator]

Shapefile

  • Croqui georreferenciado: croqui exportado em WGS 84 long-lat. [não]

DXF

  • Final aux: marcar final de medidas auxiliares com cruzes nos arquivos DXF exportados. [sim]
  • Versão: versão alvo dos arquivos DXF exportados. {6,12,16}. [6]
  • Splines: interpolação cúbica de splines DXF. [não]
  • Referência: incluir eixos X-Y nos arquivos DXF exportados. [não]
  • Camadas croquis: criar camadas nos arquivos DXF exportados. [não]

KML

  • Incluir base: incluir bases nos arquivos KML/GeoJSON/Shapefile exportados. [sim]
  • Incluir aux: incluir medidas auxiliares nos arquivos KML/GeoJSON/Shapefile exportados. [não]

GPX

  • GPX track único: exportar topos como um track único no arquivo GPX. [sim]

CSV

  • CSV completo: exportar todos os dados no CSV. [não]
  • Separ campo CSV: caracter usado para separar campos nos arquivos CSV exportados {vírgula, traço vertical, tab}. [traço vertical]
  • Final linha CSV: caracter usado para finalizar linhas nos arquivos CSV exportados {Linux, Windows} [Linux]

Dados topo

  • Equipe: equipe padrão para novas topos. Campo vem vazio por padrão. Quando você cria uma nova topo você deve inserir a equipe.
  • Nomear bases: método usado para nomear automaticamente novas bases {nenhum | aux, visada | aux, contra | contra, aux, visada | tripé | anomalia magnética | visada, aux | contra, aux | TopoRobot}. As opções disponíveis variam de acordo com o nível de recursos escolhido. [aux, visada]
  • Tipo nome base: tipos de caracteres usados para nomear bases {alfanumérico, numérico}. [alfanumérico]
  • Nome base inicial: por padrão é vazio. Nesse caso o TopoDroidX usa "0".
  • Tam miniatura: tamanho da versão miniatura das fotos. Pode variar entre 80 e 400 pixels. [200 px]
  • Base editável: nomes de bases editáveis na lista de medidas [não]
  • Origem fixa croqui: mesma origem sugerida para todos os croquis. [não]
  • Cortes compartilhados: cortes em bases compartilhados entre croquis. [não]

Unidades

  • Distância: unidade usada para visualização de distâncias {metros, pés}. [metros]
  • Ângulo: unidade usada para visualização de ângulos {graus, grados}. [graus]
  • Grade croqui: unidade usada na grade dos croquis {metros, jardas, 2 pés, 0,1 metro}. [metros]
  • Régua: unidade de comprimento da ferramenta de medição {célula grade, metros, pés}. [célula grade]

Dados medidas

  • Toler visadas: tolerância máxima aceita entre medidas consecutivas de uma mesma visada > 0%. [0,05 ou seja 5%]
  • Compr máx medida: comprimento máximo de medidas. [50 m]
  • Compr mín visada: comprimento mínimo de visadas. [30 cm]
  • Qt mín medidas/visada: quantidade mínima de medidas similares para uma nova base ser criada automaticamente {2, 3, 4}. [3]
  • Lim vert aux: ângulo máximo a partir dos 90° para que uma nova visada seja considerada como vertical no perfil retificado. [0,90]. [10°]
  • Lim inclin OLNS: inclinação da visada abaixo da qual medidas auxiliares são consideradas EDTC. Acima deste valor são considerados OLNS (Oeste-Leste-Norte-Sul) [0,90]. [80°]
  • Sent man perfil ret: sentido manual no perfil retificado (E/D) ou relativo ao azimute de referência do perfil. [relativo]
  • Botões Ant/Próx: apresentar botões Anterior/Próximo na tela de medidas. [sim]
  • Contra visada: apresentar campos de contra azimute e inclinação invertida na inserção manual de medidas. [não]
  • Aviso visada: tipo de aviso ao reconhecer nova visada {nenhum, som, vibração} [nenhum]

Localização

  • Unidade coordenada: unidade para visualização de coordenadas {graus decimais | graus, minutos, segundos}. [graus, minutos, segundos]
  • SRC: nome do Sistema de Referência de Coordenadas como usado no app Proj4. [Long-Lat, ou seja, WGS84 geográfico]
  • Altitude negativa: permitir altitudes negativas. [não]
  • Altitude editável: permitir edição de altitudes. [não]
  • Tempo loc precisa: tempo de espera para obtenção de localização precisa, em segundos.
  • App GNSS: app GNSS externa {nenhum, Mobile Topographer, GPX Recorder, GPS Position, GPSTest, GPS Logger} [nenhum]

Acurácia

  • Toler aceleração: percentual de tolerância de diferença de valores de aceleração em relação ao seu valor médio acima do qual a medida será considerada pouco confiável (somente DistoX) > 0. [1%]
  • Toler campo magnético: percentual de tolerância de diferença de valores de campo magnético em relação ao seu valor médio acima do qual a medida será considerada pouco confiável (somente DistoX) > 0. [1%]
  • Toler merg magnético: percentual de tolerância de diferença de valores de mergulho magnético em relação ao seu valor médio acima do qual a medida será considerada pouco confiável (somente DistoX) > 0. [2.0°]
  • Lim irmã: diferença percentual máxima entre os valores de visadas acima do qual são consideradas irmãs ruins > 0. [5%]

Desenho croqui

  • Arrastar canto: habilitar o arrasto do croqui pelos cantos na tela de desenho. [não]
  • Controle zoom: controle do zoom em telas multitoque {desligado, temporário, permanente}. [desligado]
  • Lim hor corte H: limite horizontal para cortes serem considerados horizontais em perfis.
  • Verif linha trena: verificar se todas as bases estão conectadas à linha de trena. [sim]
  • Verif perfil retif: verificar se todas as visadas tem uma direção no perfil retificado definida. [sim]
  • Tam barra ferramentas: tamanho da barra de ferramentas.

Pontos

  • Ícone ponto sem zoom: ícones de pontos não são afetados pelo zoom. [não]
  • Escala símbolo ponto: escala dos símbolos dos pontos. [1.2]
  • Tam rótulo: tamanho dos textos dos rótulos. [24 px]
  • Escalonar rótulos: escalonar rótulos quando alterar o nível de zoom dos croquis. [não]

Linhas

  • Espess linha: espessura das linhas no desenho. Obs.: as linhas de parede tem o dobro dessa espessura. [1 px]
  • Escala linha: escala para desenho da simbologia das linhas.
  • Estilo linha: estilo das linhas, forma como é desenhada {detalhada, normal, grosseira, Bézier (curva), simplificada}. [normal]
  • Dist mín ponto linha: distância mínima entre pontos de uma linha de tipo normal. [10 px]
  • Fechar linhas fecháveis: linhas fecháveis devem ser fechadas automaticamente. [sim]
  • Tam desnível: tamanho do desnível em linhas tipo desnível {1,20}. [8]
  • Tam marca corte: tamanho da marca indicando direção do corte.
  • Ponto corte: criar automaticamente ponto de corte atrelado a cada novo corte indicando posição onde apresentar o croqui do corte. [sim]
  • Unir linha: habilitar opção para unir linha. [não]
  • Visib borda área: visibilidade padrão das bordas de áreas. [sim]

Desenho

  • Espess linha: espessura das linhas (visadas e medidas auxiliares) {0,5; 10}. [1 px]
  • Tam nome base: tamanho dos nomes das bases. [24 px]
  • Raio ponto verde: raio dos pontos verdes no modo edição {0,5; 100}. [5 px]
  • Raio seleção: raio da ferramenta de seleção do croqui. [24 px]
  • Tam borracha: raio da ferramenta borracha. [36 px]
  • Sensib desloc: sensibilidade no deslocamento em pixels por passo. [60 px]
  • Desloc máx selec: deslocamento máximo entre início e fim de um toque na seleção. [16 px]
  • Transp aux: transparência aplicada às medidas auxiliares {0: transparente, 100: opaca}. [80%]

Dispositivos

  • Bluetooth: habilitar verificação do Bluetooth na abertura do aplicativo {desativado, verificar ao abrir app, ativado}. [ativado]
  • Tipo conexão DistoX: {sob demanda, contínua, multi}. [sob demanda]
  • Obter qt dados novos: obter a quantidade de novos dados antes de baixá-los. [não]
  • Soquete DistoX: tipo de soquete bluetooth para DistoX {normal, inseguro}. [padrão varia entre os vários tipos de dispositivos]
  • Pareamento automático: parear automaticamente após a busca com os novos dispositivos encontrados. [sim]
  • Feedback conexão: tipo de feedback ao estabelecer conexão bluetooth com dispositivo. {nenhum, som, vibrar} [nenhum]
  • Dispostivos sem nome: habilitar opção para cadastrar nomes de dispositivos. [não]

Calibrações

  • Regra agrupamento: regra padrão para agrupamento dos dados de calibração. {De 4 em 4, só os primeiros 16). [De 4 em 4]
  • Toler grupos: tolerância máxima entre medidas de um mesmo grupo. [40°]
  • Erro algoritmo: limite de erro para interrupção do cálculo dos coeficientes de calibração. [0.000001]
  • Qt iter algoritmo: quantidade máxima de iterações no cálculo dos coeficientes. [200]
  • Download dado remoto: fazer download de dados remotos de calibração imediatamente. [sim]
  • Dado bruto: exibir dados brutos. {nenhum, decimal, hex, G decimal, G hex, M decimal, M hex} [nenhum]

Visualizador 3D

  • Visão abaixo horizonte: permitir pontos de vista abaixo do horizonte. [não]
  • Ponto base: sempre mostrar pontos (verdes) representando bases. [não]
  • Raio ponto base: raio dos pontos das bases. [8 pxl]
  • Tam nome base: tamanho dos nomes das bases. [20 pt]
  • Raio seleção: raio da ferramenta de seleção. [50 pxl]
  • Mensagem distância: usar mensagem no lugar de caixa de diálogo para distância. [não]
  • Caixa diálogo base: usar caixa de diálogo no lugar de caixa de edição. [não]
  • Grade no topo: Apresentar grade sobre o modelo. [não]
  • Extensão grade: tamanho da extensão da grade ao redor da planta baixa.

DEM

  • Buffer DEM: extensão do modelo de superfície DEM ao redor da topo a ser carregada do arquivo DEM.
  • Tam máx DEM: tamanho máximo da superfície DEM em cada direção.
  • Redução DEM: modo de redução DEM. {subamostragem, cortar} [subamostragem]

Modelagem paredes

  • Uso aux: qual tipo de medida auxiliar usar {ignorar, normal, corte} [normal]
  • Todas aux: usar todas as medidas auxiliares para modelar paredes. [sim]
  • Aux projetada: projetar as medidas auxiliares em um plano normal. [não]
  • Lim aux: medidas auxiliares fora da visada a manter.
  • Partir triângulos: partir triângulos de paredes que se interceptam. [sim]
  • Ruído pontos: pequeno vetor aleatório adicionado às medidas auxiliares ao modelar paredes. [0,1]
  • Alongar paredes: distância a alongar pontos das paredes na direção das visadas.
  • Delta PowerCrust: parâmetro delta do algoritmo PowerCrust.

Especiais

Dados topo (especiais)

  • Modo mergulho: habilitar modo mergulho. [não]
  • Med aux contra visada: verificar se a primeira medida auxiliar obtida após uma nova visada é uma contra visada (somente para o método de nomeamento automático de bases aux, visada: se a primeira medida auxiliar após a visada não for uma contra visada, será destacada em violeta). [não]
  • Aux recentes: destacar medidas auxiliares recentes. [não]
  • Lim aux recente: limite de tempo após o qual as medidas auxiliares não serão mais consideradas recentes (em segundos). [30]
  • Proj vis escala: define se as visadas poderão ser escalonadas ao serem projetadas nos perfis retificados. [não]
  • Contra visada DistoX: DistoX configurada no modo "contra visada". [não]
  • Interpolar plano: habilitar função de interpolação de plano que contém visadas (direção e mergulho). [não]
  • Sensores: habilitar sensores do Android. [não]
  • Fech poligonal: compensação de erro de fechamento de poligonais {nenhuma, normal, proporcional, seletiva, por triângulos}. [nenhuma]
  • Lim poligonal: limite percentual de compensação de poligonal no modo de compensação seletiva. [1]
  • Bússola/clino Android: habilitar bússola e clinômetro do Android. [não]
  • Espera medida: tempo de espera antes de capturar uma medida através de comandos enviados pelo TopoDroidX para a trena em segundos. [10]
  • Volume bip temporizador: nível de volume do bip do temporizador de captura de medida {10, 100}. [50%]
  • Medida errada intra visada: ignorar medidas com erro. [não]
  • Renomear base aux: renomear também as bases das medidas auxiliares para mantê-las relacionadas à mesma base durante a ação "renomear bases". [não]
  • Renomear aux só próx: somente renomear medidas auxiliares subsequentes e não anteriores ao usar 'renomear grupo' na edição de medidas. [não]

Dados medida auxiliar (especiais)

  • Classes aux: habilitar classes de medidas auxiliares. [não]
  • Cores aux: sistema de coloração de medidas auxiliares {nenhum, livre, paleta}. [nenhum]
  • Lim aux planta baixa: esconder medidas auxiliares com inclinação acima deste valor na planta baixa. [80°]
  • Tracej aux: tracejamento de medidas auxiliares na planta baixa, nos perfis ou em ambos {nenhum, clino, azimute, visualização}. [nenhum]
  • Lim tracej clino: acima desta inclinação as medidas auxilaires serão tracejadas nas plantas baixas. [50°]
  • Lim tracej azimute: medidas auxiliares divergindo do azimute da visada acima deste limite serão tracejadas nos perfis. [60°]
  • Lim tracej ângulo: abaixo deste ângulo com o plano de corte a medida auxiliar será tracejada nos cortes. [60°]
  • Cor aux tracej: cor das medidas auxiliares tracejadas.
  • Cor aux ponth: cor das medidas auxiliares pontilhadas.
  • Cor aux recente: cor das medidas auxiliares recentes.

Desenho croqui (especiais)

  • Escalonar/deslocar croqui: habilitar o escalonamento e o deslocamento de croquis. [não]
  • Dividir/unir croqui: habilitar a função de divisão, cópia, contornos e união de croquis. [não]
  • Tam caneta: tamanho da caneta para permitir que a mão toque a tela ao desenhar somente com caneta. Para desabilitar essa função, defina o valor zero. [0]
  • Qt backups: quantidade de backups do croqui mantidos. [5]
  • Intervalo backup: intervalo mínimo entre dois backups. [60 s]
  • Bases gravadas: apresentar bases gravadas em laranja. [não]
  • Só visada redesenha: somente recalcular e redesenhar linha de trena quando receber uma nova visada (para modo de download contínuo de dados). [não]
  • Transformação afim: habilitar transformação afim em croquis (permite distorcer alterando proporções e posição). [não]
  • Camadas: organizar croquis em camadas (layers) {nenhuma, por símbolo, por item}. [nenhuma]
  • Tam papel milimetrado: ajuste de escala para simulação de papel milimetrado. [0]
  • Corte inclinado: criação de cortes inclinados em perfis. [não]
  • Projeção oblíqua: ângulo máximo de projeção oblíquoa.
  • Cortar fim linha: quantidade de pontos a serem desprezados nas extremidades das linhas ao usar recurso "Tam caneta" maior do que zero

Linhas (especiais)

  • Âng endireitar: ângulo máximo das quinas para transformar em segmentos de reta. [10]
  • Acur interp Bézier: acurácia da interpolação cúbica das curvas Bézier. [10]
  • Lim quina Bézier: limite ára criação de quinas nas curvas Bézier. [10]
  • Dist ponto ref: distância máxima entre segmento e ponto de referência na simplificação de linhas (em metros). [0,5]
  • Compr seg ref: comprimento máximo de um segmento de linha na simplificação de linhas (em metros). [2,0]
  • Zona tampão seg ref: largura da zona tampão em torno de segmento de linha para simplificação de linhas (em pixels). [10]
  • Colar linha/área: habilitar opções de atração e união de linhas/áreas. [não]
  • Suav/endir segmento: habilitar opções de suavizar/endireitar segmentos de linha. [não]
  • Endireitar linhas: habilitar opção para transformar linhas inteiras em retas. [não]
  • Multisseleção objeto: disponibilizar opção de seleção/edição de vários objetos do mesmo tipo (área ou linha) dentre as opções do botão "ações compostas". [não]

Dispositivos (especiais)

  • IDs BT: definir manualmente os apelidos bluetooth de dispositivos de medida.
  • Dispositivos sem nome: habilitar opção para cadastrar nomes de dispositivos.
  • Atraso conexão: tempo de espera da conexão para permitir o pareamento (em segundos). [0.1]
  • 2 DistoX: topografando com duas DistoX simultaneamente. [não]
  • Pausa dados: pausa após o download de cada dado (em milisegundos). [250]
  • Pausa dados prontos: pausa até que os dados estejam prontos para download (em milisegundos). [500]
  • Pausa Laser: pausa após comando remoto para ligar laser (em milisegundos). [2000]
  • Pausa medida: pausa após comando remoto para obter medida (em milisegundos). [2000]
  • Verif firmware: forçar verificação de integridade de firmware.
  • Modo BRIC: modo de download de dados da BRIC {só principais, todos}.
  • Visada zerada BRIC: registrar visadas com comprimento zero da BRIC. [não]
  • Índice BRIC é ID: usar índice BRIC como ID de medidas. [não]
  • Bug bit 16 SAP5: habilitar gambiarra para lidar com bug bit 16 no SAP5. [sim]

Importar/exportar (especiais)

  • Símbolos zip em arquivos: armazenar e carregar arquivos de símbolos em/de arquivos zip. [não]
  • Form dado importação: formato de dados para importação de topos {normal, mergulho}. [normal]
  • Corte ao exportar: incluir automaticamente cortes na exportação de arquivos CSV, SVG, DXF e XVI.
  • Bases: incluir automaticamente bases nos arquivos Therion (th2).
  • Contar aux EDTC: limitar quantidade de medidas auxiliares usadas como EDTC. [não]
  • Cat compart zip: incluir categoria no pedido de compartilhametno de arquivo zip. [não]

Se a opção Contar aux EDTC estiver desabilitada, todas as medidas auxiliares (dentro de certas tolerâncias) são usadas para calcular EDTC. A dimensão transversal é definida como o valor máximo das projeções das medidas auxiliares nos eixos horizontal e vertical. Medidas auxiliares dentro das tolerâncias podem ser consideradas para ambas as dimensões horizontal e vertical.

Se a opção Contar aux EDTC estiver habilitada, somente as primeiras quatro medidas auxiliares que satisfazem as tolerâncias são usadas, e uma medida auxiliar dentro da tolerância é usada somente para a dimensão vertical ou horizontal. Cada dimensão é definida como o valor da primeira medida auxiliar aceitável. Uma dimensão pode ser deixada com valor zero ao disparar uma medida auxiliar em uma direção já definida.

Por padrão as tolerâncias são muito permissivas: 0° para direções verticais e 90° para horizontais.

A opção Cat compart zip é usada para categorizar o compartilhamento de arquivos zip. Se não houver nenhum aplicativo que possa lidar com o compartilhamento, a ação falha.

Import/Export configurações

As configurações podem ser exportadas para o arquivo "settings.txt" (no diretório privado do TopoDroidX). Este é um arquivo de texto listando os valores de todas as configurações.

As configurações podem ser importadas do mesmo arquivo. É possível escolher se importar todas as configurações ou somente as funcionais.

Se houver um problema ao analisar o arquivo, as configurações não são alteradas.

Algumas configurações de interface (como o tamanho dos botões) somente se tornam efetivas após reiniciar o app.
