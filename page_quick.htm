GUIA RÁPIDO TopoDroidx

Esta página cobre as funcionalidades básicas do TopoDroidX. Consulte o Manual do Usuário para maiores detalhes.

[1] Pareando um dispositivo com o Android

O dispositivo de medição a ser usado deve ser previamente pareado com o Android. Isso deve ser feito para todos os tipos de dispositivos (DistoX, SAP5 e BRIC4). Use o app "Configurações de seu Android. Ligue seu dispositivo, vá até "rede | bluetooth" e procure por dispositivos bluetooth disponíveis. Seu dispositivo deverá aparecer na lista. Selecione-o e use o PIN "0000" (quatro zeros em sequência) se o Android sperdir um PIN. Se o seu dispositivo seguidamente pedir o PIN, troque a configuração Dispositivos >> Soquete DistoX para a opção "Inseguro".

[2] Selecione o dispostivo no TopoDroidX

Inicie o TopoDroidX. Para que o TopoDroidX trabalhe com o dispositivo desejado, clique no botão Dispostivo na tela Principal. O nome e o endereço MAC do seu dispositivo deverá ser apresentado na lista. Para selecioná-lo, clique no seu endereço de forma que passe a ser apresentado na parte superior da tela

[3] Calibre o dispositivo

Se o seu dispositivo não foi calibrado, você precisará calibrá-lo. No caso da DistoX você precisará do TopoDroidX para isso: Calibração DistoX. No caso do SAP5 e da BRIC4, isso é feito diretamente no dispositivo, sem o auxílio do TopoDroidX.

[4] Crie uma nova topo

Clique no botão Mais na tela Principal e informe o nome da topo e os nomes dos integrantes da equipe de topo na caixa de diálogo apresentada. Feche a caixa de diálogo clicando no botão Gravar.

[5] Abra a topo

Clique no nome da topo na lista apresentada na tela Principal para que seja apresentada a tela Dados topo. Este passo pode ser omitido se você clicar no botão Abrir no passo [4].

[6] Realize medições e baixe-as para o TopoDroidX

Realize as medições desejadas: vá até a primeira base, produza as medidas auxiliares e então meça a visada três vezes. Vá até a próxima base e repita esse procedimento.

Clique no botão Baixar na tela Dados topo. O download deve começar e os dados devem aparecer na lista. Os nomes das bases são criados automaticamente. Verifique que o TopoDroidX nomeou as bases corretamente. Se for necessário, corrija os nomes errados editando a medida.

[7] Crie um croqui

Cada croqui do TopoDroidX é composto por uma planta baixa e um perfil (retificado ou projetado). Clique no botão Croqui e clique no botão Novo: informe, no segundo campo, o nome da base que servirá de origem do novo croqui. Clique no botão Ok para abrir a tela Desenho com o novo croqui.

[8] Desenhando o croqui

O TopoDroidX apresenta a tela com a grade, a linha de trena, as medidas auxiliares e as bases. Esta é a planta baixa. Selecione o modo Desenhar clicando no botão Desenhar e comece a desenhar linhas. Deslique o modo Desenhar clicando novamente no mesmo botão e clique no botão Planta baixa/Perfil para visualizar o perfil. Desenhe o perfil. Feche a tela e retorne para a tela Dados topo.

[9] Termine a topo

Repita os passos a partir do [6], meça mais visadas. No passo [7], clique no nome do croqui para reabrí-lo na tela Desenhho.

[10] Crie um backup em formato ZIP

Quando tiver terminado, vá até a tela Info topo clicando na opção de menu Info topo escolha a opçao de menu Exportar e selecione o formato ZIP: dessa maneira será criado um arquivo zip com todos os dados de sua topo.

[11] Exporte os dados

Exporte os dados da topo no formato mais adequado para seu software de finalização de mapas no PC. Abra novamente o menu e clique em Exportar. Selecione o formato adequado. Para alguns formatos são apresentadas algumas opções específicas.

[12] Exporte o croqui

Voltando a tela Desenhar abra o menu e clique na opção Exportar. Selecione seu formato favorito e clique em Gravar. Para o formato cSurvey você só precisa exportar a planta baixa. Para os outros formatos é necessário exportar separadamente o perfil. Você pode precisar exportar também os cortes.

[13] Transfira os arquivos exportados

Os arquivos exportados estarão disponí9veis o subdiretório out no diretório da topo. Transfira-os para seu PC e termine seu mapa.
