Guia Rápido TopoDroidX

Esta página cobre os conceitos básicos do TopoDroidX. Consulte o manual do usuário para mais detalhes.

[1] Pareie o dispositivo de medição com o Android

O dispositivo precisa ser pareado com o Android. Isso deve ser feito também para dispositivos bluetooth Low-Energy (SAP5, BRIC 4 e 5). Faça isso com o aplicativo "Configurações" de seu Android. Vá para rede > bluetooth, ligue o bluetooth do dispositivo de medição e faça a varredura de por novos dispositivos bluetooth em seu Android. Seu dispositivo de medição deve ser localizado. Selecione-o e insira o PIN "0000" (quatro zeros) para pareá-lo com o Android, se solicitado. Se o dispositivo solicitar repetidamente o PIN, altere o item de configuração Dispositivos > Soquete DistoX para a opção inseguro no TopoDroidX.

[2] Selecione o dispositivo no TopoDroidX

Inicie o TopoDroidX. Para definir com qual dispositivo de medição irá trabalhar, clique no botão Dispositivo . O nome e o endereço do seu dispositivo devem aparecer na lista. Para selecioná-lo, toque em seu endereço, para que ele apareça no topo da tela.

[3] Calibre seu dispositivo

Se o seu dispositivo não estiver calibrado, você precisa calibrá-lo. Para o DistoX, isso pode ser feito com o TopoDroidX: consulte as instruções de calibração. Para o SAP5 e BRIC 4 e 5, isso é feito no próprio dispositivo, sem auxílio do TopoDroidX.

[4] Crie uma nova topo

Clique no botão Mais na tela Principal e insira o nome da topo e os nomes dos integrantes da equipe de topo na caixa de diálogo. Feche a caixa de diálogo clicando em "Salvar".

[5] Abra a topo

Clique no nome da topo na lista da tela Principal para abrir a tela Dados topo. Este passo pode ser pulado se você pressionar "Abrir" no passo [4].

[6] Faça as medições e baixe-as para o TopoDroidX

Faça as medições: vá para a primeira base, faça as medias auxiliares e meça a visada pelo menos três vezes. Mova-se para a próxima base e repita este procedimento. Clique no botão download na tela Dados topo. O download deve começar e os dados aparecerão na lista de medidas. Os nomes de bases são atribuídos automaticamente. Certifique-se de que o TopoDroidX não cometeu nenhum erro no nomeamento de bases. Em caso afirmativo, corrija-os editando a medição.

[7] Crie um croqui

Cada croqui no TopoDroidX é composto por uma planta baixa e um perfil. Toque no botão Croqui e depois no botão Novo: insira o nome de uma base que servirá como referência do croqui no segundo campo. Clique em OK para abrir a tela Desenho com o novo croqui.

[8] Desenhe o croqui

O TopoDroidX exibe uma tela com uma grade, a linha de trena, as medidas auxiliares e as bases. Esta é a planta baixa. Selecione o modo "desenhar" clicando no botão Desenho e comece a desenhar (linhas). Desligue o modo "desenhar" clicando nele novamente e vá para o perfil clicando no botão Tipo visualização . Desenhe o perfil. Feche a janela e retorne à lista de medidas.

[9] Complete a topo

Repita a partir do ponto [6], faça mais algumas bases. No passo [7], clique no nome do croqui para reabri-lo na tela Desenho.

[10] Faça um backup ZIP

Quando terminar, vá para a tela Info topo, clique na opção de menu Exportar e selecione o formato ZIP: isso cria um arquivo zip da sua topo.

[11] Exporte os dados

Exporte os dados da topo no formato do seu programa de mapeamento de caverna. Clique novamente na opção de menu Exportar. Escolha o formato do arquivo desejado. Para alguns formatos, você também pode escolher algumas opções de exportação.

[12] Exporte o croqui

Volte para a tela Desenho, cslique no botão Menu e selecione a opção Exportar. Escolha o formato de exportação e clique no botão Salvar. Para o cSurvey, você só precisa exportar a planta baixa. Para outros formatos, você também precisa exportar o perfil. Você pode precisar exportar também os cortes separadamente.

[13] Transferir exportações

Os arquivos produzidos na exportação são armazenadas na subpasta out dentro da pasta da topo. Transfira-os para seu computador e finalize seu mapa de caverna.
