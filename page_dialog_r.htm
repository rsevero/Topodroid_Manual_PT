Caixa de diálogo Android R

A partir do Android 11 o TopoDroidX não pode mais acessar arquivos da maneira usual por uma exigência da Google.

Os arquivos privados do app (banco de dados de dispositivos, símbolos de deesenho, arquivos de firmware, etc.) ficam armazenados no espaço privado do app ("Android/data/com.topodroid.TDX/files") onde outros apps não têm acesso. Esse diretório é apagado pelo Android quando você desinstala o app.

O banco de dados de topos e arquivos são armazenados em "Documents/TDX/TopoDroid". Este diretório não é apagado pelo Android quando você desinstala o app mas o TopoDroidX não tem acesso a ele após uma reinstalação.

Nesse caso você precisa renomear o diretório "Documents/TDX" e começar a nova instalação do TopoDroidX do zero de forma que crie um novo diretório "TDX".
