Editor de propriedades de elemento de desenho

Esta caixa de diálogo é apresentada ao se selecionar um ponto isolado ou um ponto de uma linha e clicar-se no botão Editar .

Pontos

  • Tamanho: XS (extra pequeno), S (pequeno), M (médio), L (grande), XL (extra grande)
  • Camadas: indica em quais camadas o ponto será apresentado. Disponível quando no Menu da tela Principal a opção de configuração Especiais >> Desenho croqui >> Camadas for "por item" [T]
  • Orientação: se for um ponto orientável
  • Texto do rótulo: somente para pontos do tipo rótulo
  • Opções: usado exclusivamente na exportação para o Therion

Linhas

  • Tipo de linha
  • Opções: usado exclusivamente na exportação para o Therion
  • Contorno: indica se a parte a ser tornada invisível do croqui está para fora ou para dentro da linha. Usada exclusivamente na exportação para o Therion
  • Camadas: indica em quais camadas a linha será apresentada. Disponível quando na tela Principal a opção de configuração Especiais >> Desenho croqui >> Camadas for diferente de "nenhuma" [T]
  • Direção : indica se a direção da linha é normal ou invertida. Usada exclusivamente na exportação para o Therion
  • Reta : transforma uma linha Bézier em uma polilinha, ou seja, uma linha composta de segmentos de reta consecutivos
  • Simplificar : simplifica a linha removendo pontos intermediários
  • Fechar : fecha a linha adicionando um ponto no final da linha com a mesma posição do ponto inicial

Área

  • Tipo de área
  • Borda visível
  • Camadas: indica em quais camadas o ponto será apresentado. Disponível quando na tela Principal a opção de configuração Especiais >> Desenho croqui >> Camadas for "por item" [T]
  • Simplificar : simplifica a linha da borda da área removendo pontos intermediários
  • Orientação: se o tipo de área for orientável

Bases

Veja editor de bases no croqui.

Medidas

  • Nomes das bases
  • Comentário
  • Direção no perfil retificado
  • Os seguintes indicadores (liga/desliga):
    • Duplicada : indica que esta é uma medida repetida
    • Superfície : indica que é uma medida na superfície (não deve entrar na contagem de desenvolvimento da cavidade)
    • Comentada : indica que se trata de uma medida comentada (que não será considerada)
  • Ocultar a partir da base: veja ocultando linha de trena
  • Cor: escolher cor (somente para medidas auxiliares). Essa cor personalizada é apresentada tanto na lista de medidas como também é usada para identificar a medida auxiliar no croqui. É armazenda no banco de dados. Ela é exportada no arquivo zip e lida de um arquivo zip importado. O recurso de cores de medidas auxiliares personalizadas é contralada pela configuração Especiais > Dados medida auxiliar > Cores aux. As cores personalizadas podem ser removidas através da opção de menu Cor padrão na tela Dados Topo > Info topo. [T]
