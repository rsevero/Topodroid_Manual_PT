Bases gravadas

Esta caixa de diálogo é aberta a partir do botão Base na tela Dados topo, ou clicando no botão Gravar base disponível na caixa de diálogo Editar base acessível na tela Desenho.

As "bases gravadas" são bases salvas para referência futura, por exemplo em um cruzamento, para voltar e continuar o levantamento a partir dali, ou para marcar uma possível continuação a ser verificada em uma viagem futura.

A caixa de diálogo exibe a lista de bases gravadas. Você seleciona uma clicando nela.

A base selecionada é detalhada nos campos de texto. Quando acessada a partir da tela Dados topo, o campo de texto contém a última base criada, ou a base ativa se esta estiver definida. Quando acessada a partir da caixa de diálogo de edição de base, o campo de texto contém a base selecionada.

Cada base gravada deve ter uma breve descrição (comentário) ou uma característica definida através das caixas de seleção disponíveis. É possível atribuir um geocódigo a uma base gravada.

A caixa de diálogo exibe os seguintes campos:

  • Base: nome da base selecionada.
  • Características: define características da base, se houver, através de caixa de seleção ("fixa" ou "pintada").
  • Comentário: informações sobre a base.

Botões:

  • Limpar: limpa os campos que caracterizam a base: comentário e características definidas através de caixa de seleção.
  • Geocódigo: define o geocódigo da base.
  • Gravar: grava a base no banco de dados.
  • Apagar: apaga a base do banco de dados (não recuperável).
  • Fechar: fecha a caixa de diálogo.
  • Selecionar: define a base como a base "ativa" e fecha a caixa de diálogo.
