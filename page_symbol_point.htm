Ferramentas de desenho de ponto

As ferramentas de desenho de ponto são definidas em arquivos de texto.

Três ferramentas de ponto: "usuário", "corte" e "rótulo" são definidas internamente no aplicativo. As outras são carregadas da pasta privada do TopoDroidX line. Esta pasta é populada com as ferramentas de ponto empacotadas no arquivo apk de instalação do aplicativo. Você pode excluir as ferramentas que não precisa, editar as que deseja alterar e adicionar novas que não são fornecidas no apk.

Os arquivos de definição de ferramentas de ponto contém linhas especificando as propriedades da ferramenta de desenho:

  • symbol point: linha de abertura que identifica um arquivo de definição de ferramenta do tipo ponto [obrigatória].
  • name NAME onde NAME é o nome padrão do ponto [obrigatório].
  • nomes traduzidos: por exemplo, "name-pt", "name-es", etc. [opcional].
  • th_name NAME: onde NAME é o nome Therion. O nome do arquivo deve ser o mesmo que o nome Therion, sem o possível prefixo "u:" [obrigatório].
  • color RGB (RGB em formato hexadecimal, por exemplo '0xff0000' é vermelho) [obrigatório].
  • orientation FLAG [opcional, falso se ausente, verdadeiro se '1' ou 'yes'].
  • has_text FLAG [opcional, falso se ausente verdadeiro se '1' ou 'yes'].
  • roundtrip VALUE [opcional, para SVG-roundtrip, VALUE pode ser 1 (walls_shp), 2 (walls_sym), 3 (detail_shp), 4 (detail_sym) - padrão 4].
  • level indica a quais camadas de visualização esta ferramenta está associada. 1 se ausente [opcional].
  • path inicia a definição do desenho do símbolo de ponto. Este é especificado com os seguintes comandos:
    • moveTo X Y
    • lineTo X Y
    • cubicTo X1 Y1 X2 Y2 X Y
    • addCircle X Y R: (X,Y) centro, R raio
    • arcTo X0 Y0 X1 Y1 FROM_ANGLE SPAN_ANGLE: (X0,Y0 - X1,Y1) retângulo circundante, ângulos em graus, 0 ao longo do eixo X, direção é horária
  • endpath marca o final da definição do desenho do símbolo.

Name

A propriedade "name" (que deve conter o nome do tipo de ponto em inglês) é obrigatória.

A tradução do nome em outros idiomas é opcional.

Os nomes não devem conter espaços; use o sublinhado '_' em seu lugar.

O nome Therion deve coincidir com o nome do arquivo. Isso pode causar um problema quando o nome Therion contém um hífen '-' pois este é um caractere ilegal em nomes de arquivos. Neste caso, use o caractere igual '=' no nome do arquivo, o script que cria o arquivo zip dos arquivos de símbolos substitui '=' por '-' nos nomes dos arquivos.

Os nomes Therion e de arquivos não devem conter espaços.

Orientation

Se o atributo "orientation" estiver definido como verdadeiro, o símbolo de ponto poderá ser rotacionado.

Level

O atributo "level" define o valor da flag que especifica em quais camadas os itens deste tipo de ponto devem ser apresentados.

A flag é um número que é a soma de 1 (Base), 2 (Fundo), 4 (Decoração), 8 (Cobertura) e 16 (Avulsos). Veja mais detalhes em Camadas.
