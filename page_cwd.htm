Diretórios de Projetos TopoDroidX

Para administrar projetos que envolvam diversas topos é recomendável que seja usado um diretório de trabalho dedicado para cada projeto.

O diretório padrão de projetos é "TopoDroid".

Os diretórios de projetos são criados dentro do diretório base do TopoDroidX: "Documents/TDX".

Os arquivos de símbolos de desenho e os arquivos relacionados aos dispositovos (banco de dados, firmware, descargas de memória, calibrações) são armazenados somente no diretório privado do app "files" no dispositivo primário de armazenamento: por exemplo "/sdcard/Android/data/com.topodroid.TDX/files".

As ferramentas (símbolos) de desenho em uso no projeto são configuradas no banco de dados do projeto. Quando um projeto é criado, somente os símbolos de desenho padrão estão habilitados na configuração. Outros símbolos podem ser habilitados a partir do botão Paleta na tela Principal.

Os diretórios de projetos contém somente o banco de dados de topos do projeto e os diretórios das topos.

O nome do diretório de projeto atual é uma das configurações gerais do app na tela Principal.

Alerta: futuras versões do Android podem forçar que os diretórios de projetos fiquem dentro do diretório privado do app ("files").
