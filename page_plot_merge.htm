Unir/Dividir croqui

Através da caixa de diálogo Renomear/Excluir croqui é possível renomear ou excluir o croqui aberto.

Se a configuração Especiais > Desenho croqui > Dividir/unir croqui [G] estiver habilitada, existem também botões para dividir/copiar o croqui (ou seja, extrair parte dos desenhos criando um novo croqui), incluir a apresentação de contornos de outros croquis e unir os desenhos de outro croqui no croqui atual.

Dividir croqui

Há também um botão para dividir o croqui. Isso cria um novo croqui populado com os itens selecionados do croqui atual. Para selecionar esses itens, trace uma borda em torno deles na tela Desenho.

Você pode especificar o nome do novo croqui. No entanto, sua origem é a mesma do croqui atual. Da mesma forma, ele tem o mesmo tipo de perfil que o croqui atual.

Se a tela Desenho estiver exibindo a planta baixa, a planta baixa do novo croqui é populada com os itens selecionados e a o perfil estará vazio. Se a tela Desenho estiver exibindo o perfil o comportamento é inverso.

Apenas os itens visíveis são selecionados para a divisão.

Você pode escolher se deseja mover os itens selecionados para o novo croqui ou somente copiá-los para o novo croqui.

Após esta operação, a tela Desenho exibe o novo croqui.

Unir croqui

Através desse botão uma nova caixa de diálogo é apresentada para selecionar outro croqui que terá seu contorno exibido (somente suas paredes).

Unir croqui

Se o contorno de outro croqui estiver sendo exibido, você pode mesclá-lo com o croqui atual. Os itens do croqui contornados são copiados para o croqui atual.
