Log de pacotes DistoX

Esta caixa de diálogo é aberta a partir do menu Pacotes da tela Dispositivos.

Exibe a lista de pacotes registrados. Os pacotes mais recentes estão no topo da lista.

Os itens com fundo preto são pacotes recebidos do DistoX. Aqueles com fundo mais claro são enviados ao DistoX.

A cor do texto depende do tipo do pacote:

  • branco: pacote de dados (D)
  • cinza: pacote de vetor (V)
  • marrom: pacote G (G)
  • laranja: pacote M (M)
  • verde: pacotes de comando (C)
  • amarelo: outros pacotes (X)

Os pacotes do DistoX são registrados somente se a configuração Especiais > Log pacote estiver habilitada.

O banco de dados de pacotes está na pasta de arquivos privados do aplicativo.

O diálogo tem dois botões para limpar o log dos pacotes mais antigos do que um dia ou do que uma semana, respectivamente.
