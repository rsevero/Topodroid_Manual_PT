Renomear/apagar/dividir croqui

Com este diálogo você pode mudar o nome de um croqui. Ele é aberto a partir do menu Renomear da tela Desenho.

Ele tem um campo para o novo nome do croqui, e uma caixa de texto com a estação de referência do croqui (origem do croqui).

Digite o nome do novo croqui no campo de edição, e toque no botão Aplicar para confirmar.

O botão Dividir [T] permite cortar uma parte do croqui e salvá-la como um novo croqui. Você deve escrever o nome do novo croqui na caixa de edição. Então você seleciona a parte do croqui traçando uma linha ao redor dela. Através da caixa de seleção você escolhe se deseja copiar ou mover os itens selecionados para o novo croqui.

A base de referência do novo croqui é a mesma do croqui atual.

O botão Apagar marca o croqui como "apagado". Ele pode ser recuperado com o menu recuperar da tela Dados topo.

Feche o diálogo com o botão Cancelar para cancelar.
