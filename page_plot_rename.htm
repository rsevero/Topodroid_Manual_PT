Renomear/apagar/dividir croqui

Esta caixa de diálogo é aberta a partir do menu Renomear/apagar da tela Desenho, e permite mudar o nome de um croqui.

Há um campo para o novo nome do croqui, e uma caixa de texto com a estação de referência do croqui (origem do croqui). Caso a configuração Especiais > Desenho croqui > Dividir/unir croqui [G] esteja habilitada, adicionalmente aparecem na caixa de diálogo o botão Dividir [T] e a caixa de seleção Copiar itens.

Para renomear, digite o nome do novo croqui no campo de edição, e toque no botão Aplicar para confirmar.

O botão Dividir [T] permite mover/copiar uma parte dos itens desenhados e salvá-la como um novo croqui. Você deve escrever o nome do novo croqui na caixa de edição. Através da caixa de seleção você escolhe se deseja copiar ou mover os itens selecionados para o novo croqui. Após clicar no botão Aplicar você volta para a tela Desenho e seleciona a parte do croqui traçando uma linha ao redor dela.

A base de referência do novo croqui é a mesma do croqui atual.

O botão Apagar marca o croqui como "apagado". Ele pode ser recuperado com o menu Restaurar/colar da tela Dados topo.

O botão Cancelar fecha a caixa de diálogo.
