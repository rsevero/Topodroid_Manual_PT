Ferramentas de desenho de linha

As ferramentas de desenho de linha são definidas em arquivos de texto.

Três ferramentas de linha: "usuário", "corte" e "parede" são definidas internamente no aplicativo. As outras são carregadas da pasta privada do TopoDroidX line. Esta pasta é populada com as ferramentas de linha empacotadas no arquivo apk de instalação do aplicativo. Você pode excluir as ferramentas que não precisa, editar as que deseja alterar e adicionar novas que não são fornecidas no apk.

Os arquivos de definição de ferramentas de linha contém linhas especificando as propriedades da ferramenta de desenho:

  • symbol line: linha de abertura que identifica um arquivo de definição de ferramenta do tipo linha [obrigatória].
  • name NAME onde NAME é o nome padrão da linha [obrigatório].
  • nomes traduzidos: por exemplo, "name-pt", "name-es", etc. [opcional].
  • th_name NAME: onde NAME é o nome Therion. O nome do arquivo deve ser o mesmo que o nome Therion, sem o possível prefixo "u:" [obrigatório].
  • group GROUP: especifica o grupo ao qual esta linha pertence [opcional, padrão sem grupo].
  • color RGB ALPHA (RGB e ALPHA em formato hexadecimal, por exemplo '0x00ff00' é verde, e '0xff' é totalmente opaco) [RGB obrigatório, ALPHA é opcional, se não especificado, 0xff é o valor padrão].
  • width SIZE, largura da linha em unidades da opção de configuração Desenho croqui > Espess linha [opcional, padrão 1].
  • dash PATTERN, exemplo "15 5" [padrão de traço, opcional].
  • style STYLE onde STYLE pode ser 'straight' ou 'xN' [opcional].
  • effect inicia o padrão de desenho da linha e é seguido por comandos de definição de caminho, e fechado por
  • endeffect.
  • roundtrip VALUE [opcional, para SVG-roundtrip, VALUE pode ser 1 (walls_shp), 2 (walls_sym), 3 (detail_shp), 4 (detail_sym) - padrão 3].
  • level indica a quais camadas de visualização esta ferramenta está associada. 1 se ausente [opcional].
  • endsymbol: finaliza o arquivo de definição de linha.

Name

Consulte a página Ferramenta de ponto para saber como especificar os nomes.

Group

Este atributo é usado para substituir um símbolo ausente pelo símbolo do grupo.

Dash

A diretiva "dash" especifica que a linha é tracejada de acordo com um esquema que alterna traços e espaços vazios. Por exemplo, "dash 15 5" significa que a linha é desenhada com traços de 15 unidades de extensão seguidos por espacos vazios com 5 unidades de extensão.

Style

A diretiva "style" é usada para desenhar linhas usando pontos esparsos. Por exemplo, "style x3" significa que a linha é desenhada usando um a cada três pontos. O efeito é que a linha é menos "curva". Se você deseja segmentos retos, use "style straight".

Effect

Esta opçõa é usada para definir linhas cujo desenho segue algum padrão complexo..

Level

O atributo "level" define o valor da flag que especifica em quais camadas os itens deste tipo de linha devem ser apresentados.

A flag é um número que é a soma de 1 (Base), 2 (Fundo), 4 (Decoração), 8 (Cobertura) e 16 (Avulsos). Veja mais detalhes em Camadas.
