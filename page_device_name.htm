APELIDO DISPOSITIVO

A caixa de diálogo pode ser acessada através de um clique longo no dispositivo desejado na tela Dispositivo.

Apresenta algumas informações sobre o disposito e um campo texto para se definir seu apelido.

Se a configuração de segundo dispositivo estiver habilitada, naverá um checkbox para defnir este como o segundo dispositivo (somente DistoX).

Ok grava o apelido no banco de dados e fecha a caixa de diálogo.

Se o checkbox estiver marcado o dispositivo será defiido como segundo dispositivo (somente distoX).
