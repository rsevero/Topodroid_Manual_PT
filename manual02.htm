[2] Tela Principal

A tela principal tem uma barra superior com quatro a seis botões de ações e o botão do menu "três pontos verticais" em sua direita. A lista de topos preenche a tela a partir de sua base. Esta lista estará vazia até que você crie sua primeira topo.

Na primeira vez que você iniciar o app será apresentada uma mensagem de boa vindas. Você pode revê-la a qualquer momento através da opção Sobre no menu.

Para sair da TopoDroidX, na tela principal dê um clique duplo no botão VOLTAR do hardware.

Ações

  • clique em uma topo: abre a tela Medidas Topo
  • clique longo em uma topo: abre a tela Info Topo

Botões

  • apresenta o dispositivo em uso no momento ou uma DistoX desligada se nenhum dispositivo estiver em uso.
  • cria uma nova topo
  • importa uma topo (arquivo zip ou algum dos formatos externos suportados)
  • abre a paleta de símbolos de desenho.
  • abre o visualizador 3D [T].
  • abre o Gerenciador de Projetos Therion [T].
  • edição de croqui [T]

Menus

  • Fechar: sai do TopoDroidX
  • Paleta: [A/G] abre a caixa de diálogo para carregar outros conjuntos de símbolos ou para recarregar os símbolos já instalados.
  • Sobre: a mensagem de boas-vindas do app. Se houver conexão com a internet, essa caixa de diálogo também verifica se há uma nova versão disponível.
  • Configurações: todas as configurações do app.
  • Ajuda: apresenta uma descrição breve dos botões e opções de menu disponíveis nesta tela. No canto superior direito da caixa de diálogo é apresentado um botão "livro" que abre este manual. Todas as tela do TopoDroidX contam com um menu Ajuda similar.

Configurações

  • Diretório de trabalho: [padrão "TopoDroidX"]
  • Tamanho texto: tamanho do texto dos itens de listas [padrão 20, mín 1]
  • Tamanho botões: "pequeno", "normal" (indicado para telas de 5 polegadas), "médio", "grande" (indicado para tablets de 7 polegadas), "enorme". [padrão "pequeno"]
  • Nível recursos: "básico", "normal", "avançado", "especialista", "teste". [padrão "normal"]
  • Teclado: habilita ou desabilita os teclados do TopoDroidX.
  • Sem cursor: habilita ou desabilita o cursor dos teclados do TopoDroidX. [T]
  • Manual traduzido: habilita ou desabilita o Manual do Usuário traduzido.
  • Língua: define o idioma do app.

Aviso O sistema de cache do Android pode impedir que as novas seleções de idioma sejam visualizadas para todos os textos imediatamente. Uma solução alternativa é uma reinicialização a frio (uma parada forçada seguida de reabertura do app).

A partir da caixa de diálogo de configurações da tela Principal você pode acessar as outras caixas de diálogo de configurações:

  • Importar/Exportar
  • Dados topo
  • Desenho croqui
  • Dispositivos
  • Visualizador 3D [N]
  • Especiais [A]
  • Importação/exportação de configurações [E]

Nos níveis "especialista" e "teste" é possível exportar as configurações atuais para um arquivo que será gravado no diretório do TopoDroidX.

Se o bluetooth não estiver habilitado, o app pedirá permissão para habilitá-lo. Sem bluetooth o app não pode se comunicar com os dispositivos de mensuração restando somente a opção da entrada manual de dados.

Nessa situação o botão Download da tela Medidas Topo fica desabilitado. Na tela Desenho o botão Bluetooth é desabilitado e o botão Download é substituído pelo botão Incluir.
