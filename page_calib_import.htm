IMPORTAÇÃO E EXPORTAÇÃO DE CALIBRAÇÕES DistoX

Importar calibração

Uma calibração pode ser importada a partir de um arquivo csv com dados de calibração.

A caixa de diálogo de importação pode ser acessada através do botão Importar na caixa de diálogo Calibrações.

A caixa de diálogo apresenta a lista de arquivos csv de calibração. Clique no arquivo que quiser importar.

A importação falha se o arquivo não contiver dados de calibração no formato esperado ou se se tratar de uma calibração já existente.

Pode também falhar se o endereço Blouettoth registrado no arquivo de calibração for diferente do endereço do dispositivo atual.

Exportar calibração

Uma calibração DistoX pode ser exportada em formato csv (texto).

A caixa de diálogo Exportar calibração pode ser acessada através da opção de menu Exportar na tela Info calibração.

O arquivo de calibração csv contém os detalhes da calibração (nome, data, endereço Bluetooth do dispositivo, comentário e algoritmo usado) seguido dos dados brutos e dos coeficientes calculados.

Cada linha de dados contém:

  • identificador da linha de dados
  • valores G e M (seis números inteiros)
  • grupo
  • azimute, inclinação e rotação
  • erro
  • situação
