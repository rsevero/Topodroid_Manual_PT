IMPORTAÇÃO E EXPORTAÇÃO DE CALIBRAÇÕES DistoX

Importar calibração

Uma calibração pode ser importada a partir de um arquivo csv com dados de calibração.

Esta caixa de diálogo de importação pode ser acessada através do botão Importar na caixa de diálogo Calibrações, a partir da tela Dispositivos DistoX.

A caixa de diálogo apresenta a lista de arquivos csv de calibração. Clique no arquivo que quiser importar.

A importação falha se o arquivo não contiver dados de calibração no formato esperado ou se se tratar de uma calibração já existente.

Pode também falhar se o endereço bluettoth registrado no arquivo de calibração for diferente do endereço do dispositivo atual.

Exportar calibração

Uma calibração DistoX pode ser exportada em formato csv (texto).

A caixa de diálogo Exportar calibração pode ser acessada através da opção de menu Exportar na tela Info calibração, disponível ao selecionar uma das calibrações listadas na caixa de diálogo Calibrações, a partir da tela Dispositivos DistoX.

O arquivo de calibração csv contém os detalhes da calibração (nome, data, endereço bluetooth do dispositivo, comentário e algoritmo usado) seguido dos dados brutos e dos coeficientes calculados.

Cada linha de dados contém:

  • identificador da linha de dados
  • valores G e M (seis números inteiros)
  • grupo
  • azimute, inclinação e rotação
  • erro
  • situação

O arquivo exportado será salvo na pasta Android // data // com.topodroid.TDX // files // ccsv do dispositivo Android. O conteúdo da pasta Android // data só pode ser acessado conectando o dispositivo Android a um computador e ligando o modo "Transferência de arquivos" na conexão USB do dispositivo Android.
