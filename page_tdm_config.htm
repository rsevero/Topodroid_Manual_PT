Gerenciador de Projetos - Tela Projeto

A tela Projeto é aberta ao se clicar em um nome de projeto na lista do Gerenciador de Projetos.

Exibe a lista de topos que compõem o projeto. A lista fica vazia se ainda não houver nenhuma topo adicionada ao projeto.

Cada topo pode ser selecionado para ações adicionais ao se tocar em sua caixa de seleção.

Menu

  • Fechar: fecha a janela.
  • Exportar: exporta o projeto de caverna em formato Therion ou Survex.
  • Apagar: remove o projeto de caverna.
  • Ajuda

Botões

  • inserir topos.
  • remover as topos selecionadas.
  • visualização 2D dos topos selecionados, em planta.
  • pontos geográficos: exibir os pontos geográficos da topo.
  • equates: listar os equates do projeto usados na exportação para Therion e Survex.
  • visualização 3D do projeto

O botão inserir exibe a lista de topos no banco de dados do TopoDroid. As topos são adicionadas ao projeto ao se tocar em suas caixas de seleção e confirmar com o botão OK.

Visualização 3D

O projeto é exibido na visualização 3D a partir da primeira topo. As topos seguintes são mostradas na visualização 3D apenas se estiverem (direta ou indiretamente) conectados à primeira topo.

Se várias topos estiverem georreferenciadas, todas podem ser exibidas em 3D, juntamente com as topos conectadas.

Visualização 2D

A visualização 2D exibe as topos em planta, com linha de trena e nomes das bases.

Cada topo tem uma cor diferente. Após uma determinada quantidade de topos as cores se repetem, portanto você deve exibir apenas duas ou três topos por vez. Os equates são mostrados como linhas tracejadas vermelhas, e as bases equacionadas têm um fundo redondo. O tamanho dos nomes das bases é igual ao usado na tela Desenho.

A visualização pode ser movida e ampliada. Uma topo pode ser agarrada (toque em uma base: ela ganha um fundo circular) e arrastada enquanto as outras topos não se movem. Dessa forma, você pode sobrepor topos de diferentes projetos que deseja equacionar. Quando elas se sobrepõem, circule-as tocando no ponto: você pode então inserir o equate entre elas com o botão equate .

Botões

São apresentados quatro botões:

  • Adicionar equate: abre um diálogo dizendo quais topos devem ser equacionadas. Para adicionar manualmente um equate entre topos do projeto de caverna, toque no botão OK.
  • Listar equates entre topos do projeto de caverna.
  • Reduzir número de bases identificadas
  • Aumentar número de bases identificadas

Equates

Na caixa de diálogo apresentada ao clicar no botão Listar equates você pode remover um equate clicando em sua entrada na lista.
