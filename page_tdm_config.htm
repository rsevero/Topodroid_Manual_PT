Gerenciador de Projetos - Tela Projeto

A tela Projeto é aberta ao se clicar em um nome de projeto na lista do Gerenciador de Projetos.

Os projetos de caverna administrados através desta página somente são relevantes se as topos forem exportadas para Therion ou Survex.

Exibe a lista de topos que compõem o projeto. A lista fica vazia se ainda não houver nenhuma topo adicionada ao projeto.

Cada topo pode ser selecionado para ações adicionais ao se tocar em sua caixa de seleção.

Menu

  • Fechar: fecha a janela.
  • Exportar: exporta o projeto de caverna em formato Therion ou Survex.
  • Apagar: remove o projeto de caverna.
  • Ajuda

Botões

  • inserir topos.
  • remover as topos selecionadas.
  • visualização 2D dos topos selecionados, em planta.
  • pontos geográficos: exibir os pontos geográficos da topo.
  • equates: listar os equates do projeto usados na exportação para Therion e Survex.
  • visualização 3D do projeto

O botão inserir exibe a lista de topos no banco de dados do TopoDroid. As topos são adicionadas ao projeto ao se tocar em suas caixas de seleção e confirmar com o botão OK.

Visualização 3D

O projeto é exibido na visualização 3D a partir da primeira topo. As topos seguintes são mostradas na visualização 3D apenas se estiverem (direta ou indiretamente) conectados à primeira topo.

Se várias topos estiverem georreferenciadas, todas podem ser exibidas em 3D, juntamente com as topos conectadas.
