Criar medida com sensor

Esta caixa de diálogo é apresentada a partir do botão Sensores na caixa de diálogo Edição medida.

Ela conta com uma lista de opções de sensores internos além da opção "externo" que exige a definição do tipo do sensor no campo de texto seguinte.

Há três campos de texto

  • Tipo: obrigatório quando marcada a opção "externo". O campo é preenchido automaticamente ao se selecionar um dos sensores internos.
  • Valor: atualizado automaticamente para sensores internos. Deve ser atualizado manualmente inserido para sensores externos
  • Comentário

Botões

  • Cancelar: fecha a caixa de diálog sem fazer nenhuma alteração.
  • Gravar: armazena a medida no banco de dados.

O botão Voltar do Android fecha a caixa de diálogo sem salvar nenhuma alteração.
