Pesquisa dados calibração

Esta caixa de diálogo é apresentada ao clicar-se no botão Pesquisar na tela Dados de calibração.

Permite que se pesquise por dados de calibração que excedam um valor mínimo informado no campo texto.

Os dados de calibração com erros superiores ao valor informado serão destacados com fundo laranja no campo de erro.

Botões

  • Fechar: fecha a caixa de diálogo sem fazer nada.
  • Limpar: apaga qualquer destaque.
  • Pesquisar: executa a pesquisa e apresenta os resultados.
