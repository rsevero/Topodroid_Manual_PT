Editor de pontos de linha

Esta caixa de diálogo é apresentada quando o botão Editar é clicado com um ponto de uma linha já selecionado na tela Desenho.

Propriedades

  • Tipo de linha: o tipo não pode ser alterado para "corte"
  • Opções: usado exclusivamente na exportação para o Therion
  • Contorno: "fora", "dentro" ou indefinido (quando nenhuma opção é selecionada). Indica se a parte a ser tornada invisível do croqui está para fora ou para dentro da linha. Opção relevante para a exportação no formato Therion.
  • Camadas: indica em quais camadas o ponto será apresentado. Disponível quando a opção de configuração Especiais >> Desenho croqui >> Camadas não for "nenhuma" [T]. Nela você pode alterar em quais visualizações a linha ira ser exibida. As visualizações disponíveis são representadas por letras. A linha não será apresentada se ela não estiver atribuída a nenhuma visualização. Entretanto os pontos da linhas serão exibidos no modo de edição de maneira que a linha permaneça acessível para que possa ser atribuída a alguma visualização.

Botões

  • Inverter: inverte a direção da linha. Usado na exportação para o Therion.
  • Reta: transforma uma linha curva (Bézier) em uma polilinha (uma linha composta de segmentos de reta consecutivos).
  • Simplificar: simplifica a linha apagando pontos intermediários. A quantidade de pontos das linhas pode ser reduzida apagando-se um ponto a cada dois (clique simples) ou um ponto a cada quatro (clique duplo).
  • Fechar/abrir: alterna a linha entre fechada e aberta (quando fechada, adiciona um ponto no final da linha com a mesma posição do ponto inicial).
  • Cancelar: fecha a caixa de diálogo descartando as alterações.
  • Ok: grava as alterações.

Linha de corte

Linhas de corte são especiais pois são associadas a cortes (tanto na forma de croquis quanto na forma de fotos). Dessa maneira contam com outros campos:

  • Ícone da foto: versão reduzida da foto e informações (azimute, inclinação, data e hora), se o corte contar com uma foto. Clicando-se na versão reduzida, a foto é apresentada.
  • comentário sobre o corte: campo de texto livre.
  • Quatro botões de ação:
    • Foto: ativa a câmera para captura de uma imagem. Na janela para captura de foto, 3 botões são apresentados no canto inferior direito:
      • dispara a foto;
      • inclui a foto no corte;
      • descarta a foto.
    • Croqui: abre uma nova tela de desenho para o corte.
    • Apagar: apaga o corte
    • Gravar: grava as edições no campo comentário. Este botão só é disponibilizado se há algum conteúdo no campo comentário ou se uma foto foi incluída no corte.
