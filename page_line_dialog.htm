Editor de pontos de linha

Esta caixa de diálogo é apresentada quando o botão Notas é clicado com um ponto de uma linha já selecionado na tela Desenho.

Propriedades

  • Tipo de linha: o tipo não pode ser alterado para "corte"
  • Opções: usado exclusivamente na exportação para o Therion
  • Contorno: "fora", "dentro" ou indefinido. Define a parte que faz parte da caverna. Opção relevante para a exportação no formato Therion
  • Camadas: indica em quais camadas o ponto será apresentado. Disponível quando a opção de configuração Especiais >> Desenho croqui >> Camadas não for "nenhuma" [T]

Botões

  • Inverter: inverte a direção da linha
  • Reta: transforma uma linha curva em uma reta
  • Simplifica: simplifica a linha apagando pontos intermediários
  • Quina: transforma linha suave em linha com quinas
  • Fechar/abrir: alterna a linha entre fechada e aberta
  • Cancelar: fecha a caixa de diálogo descartando as alterações
  • Ok: grava as alterações

Obs.: o segundo, terceiro e quarto botões são mutuamente excludentes, ou seja, a aplicação de um invalida a aplicação de outro.

Simplificação de linhas

A quantidade de pontos das linhas pode ser reduzida apagando-se um ponto a cada dois (clique simples) ou um ponto a cada quatro (clique duplo).

Linha de corte

Linhas de corte são especiais pois são associadas a cortes (tanto na forma de croquis quanto na forma de fotos). Dessa maneira contam com outros campos:

  • comentário sobre o corte: campo de texto livre
  • Ícone da foto: versão reduzida da foto e informações (azimute, inclinação, data e hora), se o corte contar com uma foto. Clicando-se no ícone a foto é apresentada.
  • Quatro botões de ação:
    • Foto: ativa a câmera para captura de uma imagem
    • Croqui: abre uma nova tela de desenho para o corte
    • Apagar: apaga o corte
    • Gravar: grava as edições no campo comentário. Este botão só é disponibilizado se há algum conteúdo no campo comentário.
