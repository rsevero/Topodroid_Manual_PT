Estrutura de diretórios do TopoDroidX

Esta página reune algumas informações sobre como o TopoDroidX organiza e armazena os dados das topos, os dados de calibração e os croquis.

O aplicativo cria seu diretório base padrão no armazenamento externo primário: "Documents/TDX". Este diretório não é removido quando você desinstala o aplicativo. Neste diretório, o aplicativo cria a pasta de topos "TopoDroid" com o banco de dados SQLite "distox14.sqlite" onde são armazenados os dados da topos, os arquivos de croquis das topos e outros arquivos das topos.

O aplicativo também cria subdiretórios para os diferentes tipos de arquivos.

Os subdiretórios abaixo são criados no diretório privado do aplicativo, ou seja, "Android/data/com.topodroid.TDX/files" no armazenamento externo primário.

  • bin: arquivos de firmware (DistoX v. 2 e BLE).
  • ccsv: arquivos de texto CSV da exportação de calibrações (apenas DistoX).
  • dump: arquivos de descarga de memória (apenas DistoX).
  • symbol: arquivos das ferramentas de desenho (pontos, linhas, áreas).
  • man: tradução do manual do usuário.

O diretório "bin" é usado para armazenar arquivos de firmware, tanto os arquivos de fábrica incluídos no TopoDroidX quanto os arquivos de firmware baixados de uma DistoX2.

O banco de dados "device10.sqlite" com os dados de calibração e as informações do dispositivo está também neste diretório.

Para cada topo é criada uma pasta de topo (nomeada com o nome do topo) que contém os seguintes subdiretórios:

  • audio: comentários de áudio (arquivos wav).
  • note: arquivo de texto com anotações da topo.
  • photo: fotos da topo (arquivos jpg).
  • tdr: arquivos binários que contém os croquis.
  • out: arquivos exportados.
