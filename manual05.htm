[5] DADOS CALIBRAÇÃO DistoX

Somente para DistoX, DistoX2 e DistoXBLE

A tela de dados de calibração apresenta a lista de dados de calibração. Se a calibração não tem nenhum dados, a tela estará vazia exceto pelos botões superiores. Cada linha de dados de calibração apresenta as seguintes informações:

  • ID: índice incremental que identifica a linha de dados.
  • Grupo: número do grupo a que esta linha faz parte.
  • Erro: a diferença, em graus, da direção desta linha já corrigida pela calibração e a direção média do grupo.
  • Azimute, inclinação e rotação: calculados a partir dos dados brutos sem nenhuma correção da calibração.
  • Dados brutos de calibração: opcionalmenteos componentes G e M dos dados da DistoX em valores decimais ou hexadecimais.

Cores

Os dados de calibração são apresentados com textos alternadamente em amarelo e azul para os grupos pares e ímpares. Dados sem agrupamento são cinzas.

A cor de fundo dos dados são:

  • Preto: valores normais.
  • Violeta: dados incompletos (não usados na calibração mas usados para os grupos).
  • Marrom: dados saturados.
  • Cinza: dados apagados (somente se a apresentação de dados apagados estiver habilitada).
  • Vermelho: dados com erros superiores a 1°.
  • Verde: dados diferindo da média do grupo por um valor superior ao definido em Tolerância grupos.

AÇÕES

  • clique em uma linha: apresenta a caixa de diálogo de edição de dados.

BOTÕES

  • alterna o modo de calibração da DistoX entre ligado e desligado. Pode falhar na primeira tentativa.
  • reinicia a conexão bluetooth ou permite o controle remoto da DistoX.
  • download dos dados de calibração da DistoX.
  • organiza os dados disponíveis em grupos ou reatribui números aos grupos.
  • calcula os coeficientes de calibração.
  • apresenta a distribuição de direções dos dados de calibração.
  • lê e apresenta os coeficientes de calibração armazenados na DistoX.
  • grava os coeficientes de calibração na DistoX. Este botão só é habilitado depois que os novos coeficientes de calibração foram calculados. Os coeficientes podem também ser gravados na DistoX diretamente na caixa de diálogo que apresenta os resultados do cálculo dos coeficientes. Um alerta é apresentado se a cobertura de direções dos dados for menor do que 95% ou se o erro médio for superior a 0.5°.
  • abre a caixa de diálogo de pesquisa.

No momento somente é possível o download sob demanda dos dados de calibração. Os dados baixados são adicionados à calibração atual. Desta maneira você pode realizar algumas medidas, baixá-las, verificá-las. Se quiser pode realizar mais medições e baixá-las até que você esteja satisfeito com seus dados de calibração.

Dados baixados podem ser desabilitados alterando o número de grupo da linha para "0" (zero). Dados desabilitados não são usados no cálculo dos coeficientes de calibração.

Podem também ser apagados deixando assim de ser apresentados na lista de dados da calibração. Dados apagados não são considerados no processo de agrupamento e nem no cálculo dos coeficientes de calibração.

Bluetooth

Este botão reinicia a conexão bluetooth do Topodroid.

Ele também permite que o usuário controle remotamente a DistoX [A]: ligar e desligar o laser e realizar uma medida. Os dados são imediatamente baixados se o tipo conexão DistoX for "Contínua".

Neste caso também é possível medir (e baixar os resultados) de um grupo de medidas para calibração com um pequeno intervalo entre elas. Este recurso é controlado pelas configurações "Pausa laser" e "Pausa medida".

O botão bluetooth fica desabilitado enquanto o Topodroid está realizando uma medida de calibração.

Cálcula da calibração

O botão engrenagem inicia o cálculo dos coeficientes de calibração.

Quando o cálculo é finalizado, a caixa de diálogo com os novos coeficientes é apresentada.

Se não houverem dados suficientes ou grupos de dados suficientes, o cálculo não é realizado e uma mensagem de alerta é apresentada.

O cálculo de coeficientes falha se o limite de iterações for atingido antes do processamento terminar. Isto normalmente ocorre quando os grupos não foram definidos corretamente. Uma mensagem de alerta é apresentada nesse caso.

Alertas

  • Dados saturados: dados com valores anormais de campo magnético. Uma causa possível é a presença de campos magnéticos espúrios. Dados saturados são apresentados com fundo marrom.
  • Dados zerados: dados de calibração recebidos erroneamente. Os dados tem valores zero e, por padrão, não são incluídos em um grupo apesar de serem considerados na distribuição de dados nos grupos. Dados zerados tem um fundo violeta.

MENUS

  • Mostrar/ocultar apagados: apresentar ou não os dados apagados.
  • Validar: validação desta calibração com outra.
  • configurações: as mesmas da tela calibração
