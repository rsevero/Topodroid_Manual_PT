Edição de medida de sensor

Esta caixa de diálogo é apresentada quando você seleciona uma medida de sensor e clica no botão Editar na tela Lista de sensores.

Ela exibe algumas informações sobre a medida (título, tipo, medida e valor) e um campo para alterar o comentário.

Botões

  • Apagar exclui a medida. Ela não é removida do banco de dados mas somente marcada como "apagada". Ela pode ser recuperada com um aplicativo que edite diretamente o banco de dados SQLite.
  • OK salva as alterações e fecha a caixa de diálogo
