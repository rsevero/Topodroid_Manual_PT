Gerenciador de Projetos - Criar projeto

A caixa de diálogo para criar um novo projeto é apresentada ao se clicar no botão Criar na tela Gerenciador de Projetos.

É utilizada para criar um novo projeto.

Possui apenas um campo onde deve ser informado o nome do projeto a ser criado.

Botões:

  • Cancelar - fecha a caixa de diálogo sem criar o projeto.
  • OK - cria o projeto com o nome informado.
