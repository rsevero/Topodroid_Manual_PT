RECARREGAR CROQUI

A tela é aberta através da opção de menu Recarregar disponível na tela Desenho.

O TopoDroidX mantém uma série de backups dos arquivos de croquis a partir dos quais um croqui pode ser recuperado.

Esse conjunto de arquivos é nomeado com as extensões "bck", "bck0", "bck1" e assim por diante. Eles são gravados no subdiretório "tdr". Sempre que um croqui é gravado, os arquivos de backup são rotacionados (o arquivo "bck1" é renomeado "bck2" e assim por diante; "bck" é renomeado "bck0"), o antigo arquivo "tdr" é renomeado "bck" e o croqui é gravado em um novo arquivo "tdr".

Um croqui é gravado sempre que for fechado, que a visualização for alternada entre planta aixa e perfil, for aberto um corte ou for trocado por outro croqui.

É também gravado quando é modificado enquanto está aberto (por exemplo, um novo item é incluído, ou parte de um croqui é apagado). Após uma modificação um temporizador é iniciado e quando ele expirar o croqui será salvo novamente. O tempo padrão desse temporizador é configurável (Especiais > Desenho croqui > Intervalo backup) com duração padrão de 60s.

A quantidade de backups armazenada também é configurável (Especiais > Desenho croqui > Qt backups).

A tela Backup de croqui apresenta uma pré-visualização do backup selecionado. Algumas informações sobre o backup são apresentadas na barra superior:

  • a quanto tempo o bakcup foi gravado: com sufixos para minutos (m), horas (d), meses (M) e anos (Y);
  • tamanho do arquivo em bytes entre colchetes;
  • número do arqivo de backup.

Há dois botões para mover para frente e para trás na lista de backups e o último para voltar para o croqui mais recente fechando a tela Backup de croqui.
