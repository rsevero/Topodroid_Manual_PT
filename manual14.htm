[14] Gerenciador de Projetos

Esta tela é acessada através do botão Gerenciador de Projetos disponível na tela Principal.

Um projeto reúne topos que estão no banco de dados do TopoDroid. Os projetos são armazenados em arquivos Therion no diretório "thconfig". O arquivo do projeto é atualizado sempre que um projeto é fechado. Projetos podem ser exportados como arquivos Therion ou Survex, tendo seus arquivos gravados nos subdiretórios "th" e "svx" respectivamente.

O Gerenciador de Projetos apresenta a lista de projetos disponíveis no subdiretório "thconfig". Essa lista estará vazia se você não criou nenhum projeto.

BOTÕES

  • Novo projeto: abre a caixa de diálogo para criação de um novo projeto.

MENU

  • Fechar: fecha a tela do Gerenciador de Projetos retornando para a tela Principal.
  • Ajuda

Clicando no nome de um projeto abre a tela Projeto.
