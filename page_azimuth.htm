Direção no perfil retificado

A caixa de diálogo é acessada através do botão Azimute disponível tanto na tela Medidas topo quanto na tela Desenho.

A direção no perfil retificado define como a medida será representada no perfil retificado.

A direção no perfil retificado é automaticamente definida somente para as visadas e não para as medidas auxiliares.

O botão Azimute é representado ou por uma longa seta apontando para cima (para definição automática de direção considerando um azimute de referência) ou por uma seta curta apontando para a esquerda ou direita (para direções fixas).

Quando no modo automático, a direção é definida para a direita se a diferença entre o azimute da visada e o azimute de referência for menor do que 90° subtraído o Lim vert aux (configuração disponível em Dados topo >> Dados Medidas >> Lim vert aux), e para a esquerda se for maior do que 90° somado o Lim vert aux. Será considerado vertical nos demais casos.

Quando a opção Dados topo >> Dados Medidas >> Ref fix perf retific estiver habilitada, é estabelecida uma orientação fixa para a visada no perfil e o botão de azimute só apresentará as opções esquerda e direita . Clicando no botão a orientação é alternada e as novas visadas terão suas direções no perfil retificado conforme definido.

Quando a opção Dados topo >> Dados Medidas >> Ref fix perf retific estiver desabilitada, o botão Azimute abre a caixa de diálogo Direção perfil retificado com várias opções para o ajuste:

  • rosa dos ventos: quando clicado, soma 90° à direção atual definida para o perfil retificado;
  • barra de rolagem: para definir o valor de referência variando de -180° a 180°;
  • campo de texto: para digitar o valor númerico do azimute de referência entre 0° e 360°;
  • bússola: quando clicado define o valor de referência usando a bússola do dispositivo Android;
  • esquerda: define a esquerda como direção fixa no perfil retificado;
  • direita: define a direita como direção fixa no perfil retificado;
  • Ok: grava o azimute inserido na caixa de diálogo como referência.

Direção no perfil retificado para novas visadas

Quando uma nova visada é baixada o TopoDroidX define automaticamente sua direção no perfil retificado. Isso vale também para as visadas inseridas manualmente, se nenhuma opção de direção para o perfil retificado for escolhida na caixa de diálogo Inserir medida manualmente. A direção é definida de acordo com o valor determinado nesta caixa de diálogo: Direção perfil retificado.

Direção no perfil retificado de medidas auxiliares

A direção das medidas auxiliares no perfil retificado é definida de acordo com os azimutes das visadas que chegam na base a qual a medida auxiliar está ligada. A medida auxiliar é projetada no plano vertical mais próximo passando pelas visadas que chegam na base.
