Direção no perfil retificado

A caixa de diálogo é acessada através do botão Azimute disponível tanto na tela Medidas topo quanto na tela Desenho.

A direção no perfil retificado define como a medida será representada no perfil retificado.

A direção no perfil retificado é automaticamente definida somente para as visadas e não para as medidas auxiliares.

O botão Azimute é representado ou por uma longa seta apontando para cima (para definição automática de direção considerando um azimute de referência) ou por uma seta curta apontando para a esquerda ou direita (para direções fixas).

Quando no modo automático, a direção é definida para a direita se a diferença entre o azimute da visada e o azimute de referência for menor do que 90° menos o Lim vert aux (configuração disponível em Dados topo > Dados Medidas > Lim vert aux) e para a esquerda se for maior do que 90° mais o Lim vert aux e vertical no restante dos casos.

Quando a opção Dados topo > Dados Medidas > Ref fix perf retific estiver habilitada, o botão de azimuite só apresentará as opções esquerda e direita . Clicando no botão as duas opções são alternadas. As novas visadas terão suas direções no perfil retificado definidas de acordo.

Quando a opção Dados topo > Dados Medidas > Ref fix perf retific estiver desabilitada, o botão Azimute abre a caixa de diálogo Direção perfil retificado:

  • azimute: referência para definição automática de direção no perfil retificado
  • barra de rolagem para definir o valor de referência variando de -180° a 180°
  • campo de texto: para definir o valor númerico do azimute de referência entre 0° e 360°
  • bússola: define o valor de referência usando a bússola do dispositivo Android
  • esquerda: define a esquerda como direção fixa no perfil refiticado
  • direita: define a direita como direção fixa no perfil refiticado
  • Ok: usa o azimute atual como referência

Direção no perfil retificado de visadas

Quando uma nova visada é baixada o TopoDroidX define automaticamente sua direção no perfil retificado. Isso vale também para as visadas cadastradas manualmente se nenhuma opção de direção no perfil retificado na caixa de diálogo de cadastro manual de visada for escolhida. A direção é definida de acordo com as opções escolhidas na caixa de diálogo Direção perfil retificado.

Direção no perfil retificado de medidas auxiliares

A direção no perfil retificado de medidas auxiliares é definida de acordo com as direções das visadas que chegam na base a qual a medida auxiliar está ligada levando em consideração o azimute das mesmas. A medida auxiliar é projetada no plano vertical mais próximo passando pelas visadas que chegam na base.
