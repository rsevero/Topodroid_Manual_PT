[4] TELA CALIBRAÇÃO DistoX

Somente para DistoX, DistoX2 e DistoXBLE

Ao clicar no botão Calibração da tela Dispositivos é apresentada a lista de calibrações disponíveis para o dispositivo selecionado. Você pode abrir uma calibração pré-existente ou criar uma nova. Em todos os casos você será levado para a tela calibração.

A tela Calibração tem campos para:

  • Nome: cada calibração deve ter um nome único.
  • Data
  • Descrição
  • Inclinação magnética: em graus, se calculada pelo algoritmo de calibração
  • Algoritmo: seleção do algoritmo de calibração. A opção "auto", onde o Topodroid escolhe qual algoritmo usar, vem selecionada por padrão.

e campos rádio para seleção do algoritmo de calibração. A opção "auto", onde o TopoDroid escolhe qual algoritmo usar, vem selecionada por padrão.

BOTÕES

  • grava os dados da calibração no banco de dados.
  • apresenta a tela Dados Calibração.
  • apresenta os coeficientes de calibração armazenados no banco de dados.

MENUS

  • Exportar a calibração.
  • Apagar a calibração e seus dados do banco de dados (estão ação não pode ser desfeita).
  • Configurações
  • Ajuda

CONFIGURAÇÕES CALIBRAÇÃO (somente para a DistoX)

  • Regra agrupamento: como agrupar os dados de calibração em grupos [padrão "De 4 em 4"]
  • Tolerância grupos: limite a partir do qual os dados de calibração com grande distância angular para a média do grupo serão destacados [padrão 40 mín 0]
  • Erro: o cálculo dos coeficientes de variação termina se a variação de todos os coeficientes em uma iteração é menor do queeste valor [padrão 10-6, mín 0].
  • Iterações algoritmo: quantidade máxima de iterações do algoritmo de calibração [padrão 200, mín 50].
  • Download dados remotos: fazer download de dados remotos decalibração imediatamente.
  • Dados brutos: por padrão o TopoDroid apresenta o azimute, a inclinação e a rotação. Ative essa opção se quiser que sejam também apresentados os dados brutos da calibração (os componentes G e M), sejam em decimal ou hexadecimal. [padrão "nenhum"]
  • Algoritmo: pode ser 0 (Automático), 1 (Linear) ou 2 (Não linear). Se a opção "tipo de algoritmo" de uma calibração é deixada em "Automático" e esta configuração não estiver em "Automático" o algoritmo escolhido por esta configuração será utilizado para calcular a calibração. Se ambos estiverem em "Automático" o algoritmo a ser utilizado será escolhido automaticamente pelo TopoDroid levando em consideração o modelo e versão de firmware da DistoX. [padrão "Automático"]
