Apelido Bluetooth

Esta janela permite ao usuário definir um apelido para um dispositivo bluetooth conhecido.

O modelo do dispositivo deve ser selecionado entre as opções disponíveis. É importante selecionar o modelo correto, pois isso determina como o TopoDroidX troca dados com o dispositivo.

A menos que o dispositivo seja uma DistoX antiga (primeira versão baseada no Disto A3), o código do dispositivo deve ser inserido. Pode ser um número de quatro dígitos, ou conter caracteres. O TopoDroidX não verifica o código, mas é aconselhável usar o código relatado pelo dispositivo.

O apelido do dispositivo deve ser inserido conforme relatado na listagem do Android dos dispositivos bluetooth pareados.

Botões

  • Adicionar: Adiciona um novo apelido ou altera um apelido existente.
  • Fechar: Fecha a janela.

A janela contém também uma listagem dos apelidos bluetooth. Para remover um apelido clique na entrada da lista e confirme a remoção.
