Geopontos

Esta janela é apresentada ao clicar-se no botão GNSS na tela Info topo.

Esta janela apresenta a lista de pontos georreferenciados. O TopoDroidX não utiliza os dados de georreferenciamento nos croquis durante as sessões de desenho. As informações de georreferenciamento somente são utilizadas na exportação para formatos externos que suportem coordenadas geográficas.

Há algumas formas diferentes de incluir um ponto georreferenciado:

  • GNSS: obtém as coordenadas de um novo ponto com o GNSS (se a permissão "Localização" tiver sido concedida para o TopoDroidX)
  • Adicionar: cadastrar manualmente as coordenadas de um novo ponto
  • Importar: importa as coordenadas de um novo ponto do app GNSS disponível

ALERTA: a acurácia das coordenadas obtidas através do GPS de um telefone frequentemente tem um erro na casa do 15 a 20 m.

Clicando em um ponto georreferenciado da lista abre a caixa de diálogo Edição coordenadas.

O TopoDroidX armazena internamente as coordenadas em graus decimais. Nas caixas de diálogo de coordenadas, você pode digitá-las tanto em graus decimais ("gg.gggggg") quanto no formato "gg:mm:ss.ss". O teclado do TopoDroidX tem teclas especiais tanto para grau (°) quanto para minuto ('). Se você estiver usando um teclado que não disponibilize os dois pontos (:), use um espaço no lugar. Se não tiver o ponto decimal (.), use uma barra (/) ou uma vírgula (,).

Apps GNSS externos

O TopoDroidX pode importar coordenadas geográficas de diversos apps:

  • Mobile Topographer: arquivo de pontos. Uma caixa de diálogo é aberta para que seja escolhido o ponto a ser importado.
  • GPX Recorder: arquivo de trilha no formato GPX. O último ponto da trilha é importado.
  • GPS Position: arquivo de ponto em formato CSV. O TopoDroidX calcula a média das posições dos pontos.
  • GPS Test: coordenadas em um aqruivo de log. O TopoDroidX usa a última coordenada.
  • GPS Logger: arquivo de trilha em formato TXT. O TopoDroidX usa o último ponto da trilha.

O nome do ponto (base) importado a partir do GPX Recorder, GPX Position e GPS Test é definido automaticamente pelo TopoDroidX como um número com o prefixo "#".

Mobile Topographer: os arquivos de listas de pontos são armazenados no diretório MobileTopographer/pointlists.

GPS Recorder: os arquivos de trilhas são armazenados no diretório gpx_recorder.

GPS Position: os arquivos são armazenados no dispositivo principal de armazenamento do celular.

GPS Test: a criação de logs deve estar habilitada no app. É suficiente que se inclua nos logs as coordenadas. O arquivo de log é gravado quando o app é fechado e é armazenado no diretório gnss_log. Além disso as coordenadas geográficas gravadas na área de armazenamento pelo app GPS Test podem ser transferidas para os campos da caixa de diálogo Inserir coordenadas. Verifique que a opção para armazenar também a altitude na área de armazenamento está habilitada no GPSTest.

GPS Logger: os arquivos de trilhas devem ser exportados no formato TXT. O TopoDroidX usa o elipsóide de altitude e a acurácia.

Dicas de uso de apps GNSS (procure na internet por detalhes e instruções de uso):

  • Habilite "force full GNSS measurement"
  • Se disponível, desabilite "kinematic Kalman filtering"
  • Se disponível defina localização "static mode"
  • Se disponível, desabilite "A-GSP". Desabilite "mobile data" e limpe os dados A-GPS
  • Desabilite localização via wifi e bluetooth

Observação: desabilitar a otimização de bateria para o app só é necessário se você precisar prevenir que o sistema feche o app quando ele não estiver em primeiro plano.

Configurações de localização:

  • Dados topo >> Localização:
    • Unidade coordenada: unidade para visualização de coordenadas (graus decimais ou graus, minutos, segundos) [A]
    • SRC: nome do Sistema de Referência de Coordenadas como usado no Proj4 [long-lat]
    • Altitude negativa: permitir altitudes negativas [não]
    • Altitude editável: permitir edição de altitudes [não]
    • Tempo loc precisa: tempo de espera para localização precisa em segundos [60s]
    • App GNSS: app GNSS usada (nenhuma | Mobile Topographer | GPX Recorder | GPS Position | GPSTest | GPS Logger) [nenhuma]
