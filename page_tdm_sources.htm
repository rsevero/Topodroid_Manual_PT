Gerenciador de Projetos - Topos

Esta caixa de diálogo é apresentada ao se clicar no botão Inserir na tela Projeto.

A caixa de diálogo lista as topos existentes no banco de dados do TopoDroidX.

Para adicionar uma ou mais topos ao projeto, marque a caixa à esquerda delas e clique no botão OK.
