MAPA FINAL

Esta página descreve o uso do Therion e do cSurvey para produzir o mapa final.

Therion

Se você quiser compilar o projeto no Therion você deve produzir um arquivo thconfig que pode ser simples como:

source my_survey.th
export map -proj plan -o my_cave_p.pdf
export map -proj extended -o my_cave_s.pdf

Para isso você deverá antes ter exportado os dados da topo como um arquivo ".th" do Therion e os croquis como arquivos ".th2".

O arquivo de dados da topo contém os comandos para ler os aquivos dos croquis. Entretanto eles estão comentados exigindo que você os descomente para que sejam efetivamente lidos.

O arquivo de dados contém também comandos que definem um mapa para cada croqui (scrap). Para topos simples você pode deixar esses comandos comentados.

Alternativamente você pode exportar os croquis como arqivos ".xvi" do xTherion e redesenha-los lá.

cSurvey

Exporte os roquis como arquivos ".csx" do cSurvey.

Abra o arquivo exportado com o cSurvey: você terá uma planilha de dados preenchida com os dados da topo e os croquis alinhados adequadamente na área de desenho do cSurvey.
