Apps auxiliares

Ao trabalhar com o TopoDroidX pode ser útil usar alguns apps auxiliares:

  • Proj4: é um app muito simples, usado pelo TopoDroidX para converter coordenadas long-lat para outros sistemas.
  • GNSS: sigla para Sistema Global de Navegação por Satélites (ou Global Navigation Satellite System em inglês). Você pode usar um app deste tipo para obter coordenadas geográficas de pontos de interesse e importá-las no TopoDroidX.
  • CrowdMag: é um app do NOAA (National Geophysical Data Center ou Centro de Dados Geofísicos dos EUA) útil para obter informações atualizadas da situação do campo magnético da Terra.
  • Camera: um app para capturar imagens.
  • Administrador de arquivos: escolha um bom administrador de arquivo como o MiXplorer para visualizar arquivos e fazer backups.
  • SQLiteViewer: ou qualquer outro app para administrar bancos de dados SQLite pode ser útil para inspecionar os bancos de dados do TopoDroidX.

Apps GNSS

  • MobileTopographer: o TopoDroidX é capaz de ler as coordenadas diretamente de um arquivo de pontos do MobileTopographer.
  • GPX Recorder e GPS Position: o TopoDroidX é capaz de ler trilhas (arquivos gpx e csv respectivamente) e usar as últimas coordenadas de pontos.
  • GPSTest: o TopoDroidX é capaz de ler coordenadas de um arquivo de log do GPSTest. As coordenadas podem também ser arquivadas na área de armazenamento temporária e coladas na caixa de diálogo inserção de coordenadas em uma base topográfica.
  • GPS Point: o TopoDroidX é capaz de ler as coordenadas exportadas para um arquivo de pontos em formato texto-simples.

Além disso, a depender de como você finaliza suas topos, pode ser útil instalar:

  • um visualizador/editor de imagens (os dispositivos Android costumam vir com algum pré-instalado);
  • um visualizador/editor de arquivos DXF;
  • um visualizador/editor de planilhas eletrônicas;
  • um visualizador SVG;
  • um compressor ZIP;
  • GoogleEarth;
  • um visualizador de arquivos de trilhas.

Proj4

O TopoDroidX armazena as coordenadas de pontos no formato long-lat (WGS84). As altitudes são armazenadas tanto em relação ao elipsóide como ao geóide.

O app Proj4 é usado pelo TopoDroidX de duas maneiras para transformar coordenadas para outros sistemas:

  • converter coordenadas do formato long-lat (WGS84) para outro sistema.
  • inserir coordenadas de um ponto em um outro sistema que não o long-lat (WGS84).
