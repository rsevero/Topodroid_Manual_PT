Gerenciador de Projetos - Geopontos

Esta caixa de diálogo é apresentada ao se clicar no botão GNSS na tela Projeto.

A caixa de diálogo lista as informações de geolocalização das topos do projeto:

  • nome
  • base
  • coordenadas lat-long WGS84
  • sistema de coordenadas adicional
  • coordenadas no sistema de coordenadas adicional

Topos sem geolocalização não são listadas.
