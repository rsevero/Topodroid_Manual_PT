Mapa de cores PocketTopo

Esta caixa de diálogo é aberta selecionando-se a opção de configuração Importar/Exportar > Importar > Mapa cor PocketTopo.

Ela exibe uma tabela com três colunas:

  • cor do PocketTopo
  • tipo de linha TopoDroid associada (nome Therion)
  • tipo de ponto TopoDroid associado (nome Therion)

Os nomes dos símbolos usados no mapa de cores são os nomes usados pelo Therion para identificar os tipos de símbolos, ou seja, os nomes dos arquivos de símbolos.

Alguns símbolos são fixos (não editáveis): usuário, parede (linha), água (área) e corte (ponto e linha).

O símbolo rótulo é fixo. Ele não pode ser usado no mapa de cores porque é renderizado com um texto.

O mapa de cores é usado na importação de arquivos ".top", (formato usado pelo PocketTopo), para mapear as cores do PocketTopo aos tipos de símbolos de desenho do TopoDroidX.
