Novo croqui

Esta caixa de diálogo é apresentada ao clicar-se no botão Novos na caixa de diálogo Desenhos que, por sua vez, é apresentada ao clicar-se no botão Croquis na tela Dados topo.

Tem dois campos de texto e um botão de confirmação:

  • Nome: usado para identificar o croqui.
  • Base referência
  • Perfil projetado: por padrão o perfil é retificado. [E]

Você pode aceitar o nome sugerido pelo TopoDroidX (um número interiro incremental) ou entrar um nome mnmônico. Cada croqui é salvo em dois arquivos: um para a planta baixa e outro para o perfil. Os nomes dos arquivos são resultado da composição do nome da topo, do croqui e de um sufixo ("p" para planta baixa e "s" para perfil). Por exemplo, o croqui "2" da topo "caverna" é gravado nos arquivos: "caverna-2p.tdr" e "caverna-2s.tdr".

O TopoDroidX apresentará um alerta se o nome escolhido coincidir com o nome de um croqui existente. Não é possível criar dois croquis cujos nomes se diferenciem somente por letras maiúsculas X minúsculas pois os nomes dos arquivos desses croquis seriam tratados como iguais no sistema de arquivos insensível à caixa alta/baixa.

ALERTA

O nome do croqui não pode conter o caracter '/' (barra) pois é o caracter usado pelo Android como separador de diretórios. Barras são substituídas por hífens '-'.

Base referência

A base referência (origem) define como o croqui é transladado em relação à linha de trena da topografia. A linha de trena define a escala do croqui (o desenho é feito na escala de 40 unidades da tela por metro) e sua rotação, mas deixa indefinido como o croqui é transladado em relação à linha de trena. Essa incerteza é sanada definindo que a base de referência fica no ponto (0, 0) em relação ao espaço de desenho do croqui.

A base de referência também determina a redução de dados da linha de trena para o croqui. Do ponto de vista do croqui a linha de trena é uma linha que se inicia na base de referência e continua enquanto houver medidas que possam ser ligadas a ela. A linha de trena resultante forma uma árvore, com raiz na base de referência. Para deixar a tela menos poluída, é possível ocultar a exibição de parte da linha de trena, seja após uma base, seja antes dela (ou seja, entre a base de referência e a base escolhida). Isso é explicado na caixa de diálogo Base croqui.

Se a configuração Origem fixa croqui estiver habilitada, o TopoDroidX preenche automaticamente o campo Base referência com a primeira base da topo. Do contrário, ele usa a última base criada ou a base ativa se esta estiver definida. Você pode alterar esse valor se quiser que outra base sirva de referência para o novo croqui.

O TopoDroidX sinaliza um erro se a base referência não existir nos dados da topografia. Pode-se forçar o uso de uma base inexistente marcando a caixa de seleção Croqui solto. [T] Nesse caso o croqui não terá base de referência, nem medidas auxiliares nem bases até que uma visada seja criada com a base de referência.

Se a caixa de seleção Perfil projetado estiver marcada o perfil do croqui será do tipo projetado. A direção do perfil, ou seja, o azimute da projeção, é definido através de uma caixa de diálogo gráfica que permite rotacionar a linha de trena em torno de um eixo vertical centralizado na base de referência. Se a caixa de seleção não estiver marcada o perfil é retificado e não é apresentada a caixa de diálogo para definição do azimute da projeção.
