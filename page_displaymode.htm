DADOS TOPO: O QUE APRESENTAR

A caixa de diálogo pode ser acessada através do botão Quais medidas apresentar na tela Dados topo.

  • IDs medidas
  • Medidas auxiliares: medidas com uma só base.
  • Últimas medidas auxiliares: medidas auxiliares mais recentes. Só disponível se a opção de configuração Especiais >> Dados topo >> Aux recentes estiver habilitada. Tempo definido através da configuração Especiais >> Dados topo >> Lim aux recente em segundos.
  • Medidas vazias: medidas sem bases.
  • Visadas repetidas: medidas repetidas de uma mesma visada.

Os IDs das medidas são únicos, crescentes e atribuídos pelo TopoDroidX às medidas da topo.

Mesmo que as medidas auxialires estejam ocultas, é possível visualizar as medidas auxialires de uma base específica. Clique no nome da base em uma visada para que as medidas auxiliares dessa base sejam apresentadas. Clicando novamente no nome da base (mesmo em uma medida auxiliar) volta a ocultar as medidas auxiliares.
