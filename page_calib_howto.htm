COMO CALIBRAR A DistoX

A DistoX precisa ser calibrada antes de ser usada. A calibração é uma transformação dos dados brutos fornecidos pelos sensores da DistoX que corrige as imperfeições de alinhamento dos sensores, seus desalinhamentos com o laser e os efeitos dos componentes eletrônicos no campo magnético.

Para calcular os coeficientes dessa transformação você precisa realizar uma série de medidas para calibração "acuradas" registrando os dados brutos dos sensores. isso é feito com a DistoX no modo Calibração.

Os dados para calibração devem ser capturados em um lugar livre de influências magnéticas, ou seja, dentro de uma caverna ou no meio do mato. Fique atento a objetos metálicos e aparelhos eletrônicos que possam afetar a DistoX, especialmente os que você estiver eventualmente carregando.

Os dados devem cobrir todas as direções e para cada direção você deve capturar quatro medidas rodando a trena em torno do eixo do laser.

A melhor forma de fazer isso é criar 14 grupos de quatro medidas cada onde oito deles apontam para os vértices de um cubo imaginário que circunde o espaço de calibração e seis apontem para os centros das faces deste mesmo cubo.As quatro medidas em cada grupo devem ser realizadas rodando a DistoX 90° entre cada uma. Você pode capturar mais de 14 grupos; quanto mais melhor.

No mínimo você deve capturar quatro grupos de dados (com quatro medidas cada cada uma delas com uma rotação diferente) em quatro direções no plano horizontal: saindo do centro do cubo para o centro de cada uma das quatro faces verticais do cubo. As outras 40 medidas não precisam ser realizadas com tanto cuidado se você usar o agrupamento de dados PocketTopo porque cada uma delas será tratada individualmente. Você pode capturar mais de 40 medidas individuais se quiser.

Os dados de calibração que vão formar um grupo devem ser capturados com cuidado. Dados imprecisos de um grupo resultam em uma calibração ruim. Desta maneira é recomendável que você use pontos fixos como alvos (um ponto na parede ou em uma árvore) e distantes entre si alguns metros de maneira que a acurácia do laser seja superior a acurácia desejada para a calibração (2 cm a 5 m de distância representam cerca de 0,2°).

Após capturar os dados de calibração, baixe esses dados para o TopoDroidX. Você não precisa ter todos os dados de calibração na DistoX para baixá-los: ao baixar alguns dados eles são adicionados aos dados que eventualmente já estejam no TopoDroidX. Desta maneira, se por engano você fizer cinco capturas no lugar das quatro desejadas, baixe-as e apague a errada no TopoDroidX.

Antes de calcular os coeficientes de calibração é necessário informar o TopoDroidX como os dados de calibração devem ser agrupados. Os grupos são identificados por números positivos. Dados com o número de grupo zero ou nenhum grupo não são usados para o cálculo da calibração. Dados apagados tem o grupo -1 e normalmente não são apresentados na lista.

O TopoDroidX possue duas políticas diferentes para agrupamento automático de dados de calibração:

  • TopoDroidX: todos os dados são agrupados em grupos de 4 medidas. Está é a política padrão de agrupamento.
  • PocketTopo: quatro grupos de quatro medidas seguidos de dados individuais (não agrupados).

Os números dos grupos pordem ser atribuídos ou alterados editando os dados de calibração. Clique em uma linha de dados na lista para abrir a caixa de diálogo de edição de dados.

Após atribuir grupos aos dados é possível calcular os coeficientes de calibração. Além dos coeficientes de calibração, a caixa de diálogo apresentada ao final do cálculo apresenta também o erro médio (e o "delta" do algoritmo original), o desvio padrão do erro e o erro máximo. Além disso há um histograma com a distribuição dos erros dos dados de calibração. Esse gráfico tem duas linhas verticais: uma marca amarela a 0,5° e uma vermelha a 1,0°.

Em resumo, para calibrar a DistoX você:

  • ative o modo "calibração" na DistoX
  • capture os dados de calibração
  • baixe os dados para o TopoDroidX
  • atribua grupos aos dados (automatica ou manualmente)
  • calcule os coeficientes de calibração
  • envie os coeficientes para a DistoX
  • volte a DistoX para o modo "normal"
