Nova topo

Esta caixa de diálogo é apresentada ao clicar-se no botão Adicionar na tela Principal.

Informações sobre a nova topo:

  • Nome
  • Data
  • Equipe
  • Base inicial
  • Declinação magnética
  • Descrição
  • Cortes não compartilhados
  • Modo mergulho de dados [T]

O nome é obrigatório. Os nomes das topos são usados para listar as topos na Tela principal e o TopoDroidX distingue as topos pelo nome (embora cada topo tenha um id numérico único no banco de dados).

O nome deve ser único no banco de dados. Nomes que diferem somente pela caixa de letras não são diferentes porque o sistema de arquivos é insensível à caixa de letras. O TopoDroidX apresenta um alerta se o novo nome de topo coincide com o nome de uma topo existente exceto pela caixa de letras.

ALERTA

O nome da topo não pode conter espaços, nem os caracteres '/' (barra, que é o separador de diretórios), ':' (dois pontos), '*' (asterisco) e '\' (barra invertida). Barras invertidas são eliminadas. Espaços são substituídos por '_' (sublinhado), barras e dois pontos por hífens '-', e asteriscos por '+' (mais).

A data é escrita no formato "aaaa.mm.dd". Por exemplo, 6 de janeiro de 2015 é "2015.01.06". É pré-definida para a data atual.

O campo equipe é obrigatório e não pode ser deixado em branco.

A descrição pode ser deixada vazia mas é uma boa prática escrever uma descrição para a topo.

A declinação magnética é "não definida" por padrão. Se você sabe a declinação magnética você pode entrar aqui, caso contrário você pode procurar na web depois de ter encontrado as coordenadas de um ponto fixo (digamos, a entrada da caverna) com o GNSS [A]. A declinação deve ser inserida em graus decimais. O intervalo aceito de valores é [-360°, 360°]. Qualquer valor fora deste intervalo é considerado como "não definido".

Se a caixa de seleção Cortes não compartilhados estiver marcada, os cortes de seção transversal (x-sections) não são compartilhados entre os croquis da topo. Se a caixa de seleção não estiver marcada, os cortes de seção transversal são compartilhados entre os croquis da topo. Esta escolha deve ser feita quando a topo é criada e não pode ser modificada posteriormente porque afeta os croquis existentes.

O modo mergulho de dados é um recurso experimental. Os dados podem ser inseridos somente manualmente, e no seguinte formato:

  • Base DE
  • Base PARA
  • Profundidade na base DE
  • Azimute da direção DE-PARA
  • Distância entre as bases DE e PARA
  • EDTC na base DE (opcional)

No modo mergulho não são aceitas contravisadas.

Botões

  • Salvar: a nova topo é adicionada ao banco de dados.
  • Abrir: salva e abre a tela Dados medidas imediatamente.
